# 第一章 晶体结构

## 1.1 晶体与非晶体

### 基本概念

- **单晶**：原子（离子或分子）在整个材料中呈严格的周期性排列。**长程有序**，整体具有一致的取向。物理性质一般表现为**各向异性**。不存在晶界，整个晶体是一个连续的晶格。
- **多晶**：由大量取向随机的小单晶（晶粒）组成。每个晶粒内部原子排列有序，但晶粒之间存在晶界，晶粒间取向不同。宏观上往往表现为**各向同性**（晶粒取向随机平均的结果）。
- **非晶**：原子排列**无长程有序**，只保留局部的短程有序（如最近邻配位数、键长等与晶体相近）。无固定熔点。电子衍射图呈现弥散的晕环而非锐利的斑点。

### 晶体区别于非晶的本质特征

晶体的定义性特征是**平移对称性**——存在三个不共面的基矢 $\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$，使得将晶体平移 $\mathbf{R} = n_1\mathbf{a}_1 + n_2\mathbf{a}_2 + n_3\mathbf{a}_3$ 后与自身完全重合。这一性质决定了：
1. 原子排列的周期性 → X射线/电子/中子衍射出现尖锐的布拉格峰
2. 存在明确的熔点
3. 物理性质可以用能带理论等基于周期性的理论描述

## 1.2 布拉菲格子与原胞

### 布拉菲格子的定义

**布拉菲格子**（Bravais lattice）是由空间点阵的平移对称性定义的抽象几何点集。全部格点可表示为：

$$\mathbf{R} = n_1\mathbf{a}_1 + n_2\mathbf{a}_2 + n_3\mathbf{a}_3 \quad (n_1,n_2,n_3 \in \mathbb{Z})$$

其中 $\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$ 为**基矢**（primitive vectors）。

布拉菲格子的**根本性质**：所有格点完全等价——每个格点周围的环境（近邻原子数、方向、距离）完全相同。这是平移对称性的直接结果。

### 晶胞与原胞

**原胞（primitive cell）**：能够通过平移操作无重叠地铺满整个空间的最小体积单元。每个原胞恰好包含**一个格点**。

原胞体积：$\Omega = |\mathbf{a}_1 \cdot (\mathbf{a}_2 \times \mathbf{a}_3)|$

**原胞的构造不唯一**——任何以格点为顶点、体积等于 $\Omega$ 的平行六面体都可以作为原胞。

**晶胞（conventional cell）**：为了更直观地反映晶体的对称性而选取的重复单元，通常比原胞大。晶胞的基矢不一定是最短的平移矢量，因此一个晶胞可能包含多个格点。

例：FCC的晶胞为立方体（含4个格点），原胞为菱面体（含1个格点）；BCC的晶胞为立方体（含2个格点），原胞为菱面体（含1个格点）。

### Wigner-Seitz原胞

**构造方法**：
1. 选定一个格点为原点
2. 连接原点到所有近邻格点
3. 作这些连线的垂直平分面（三维）或垂直平分线（二维）
4. 这些面（线）围成的最小封闭区域即为WS原胞

**特点**：
- 恰好包含一个格点
- 保留了晶格的全部点群对称性
- 原胞内每一点到原点格点的距离 ≤ 到其他格点的距离
- WS原胞在倒空间中的对应物即为**第一布里渊区**

### 实际晶体结构 = 布拉菲格子 + 基元

晶格点阵（布拉菲格子）仅是描述平移对称性的抽象点阵。实际晶体在每个格点上附加一组原子（基元）。基元可以包含一个或多个原子，不同种类的原子，且原子在基元内有特定的相对位置。

$$\text{晶体结构} = \sum_{\mathbf{R}} \text{基元}(\mathbf{r} - \mathbf{R})$$

**为什么只有14种布拉菲格子，而实际晶体结构有成千上万种？**
- 布拉菲格子受平移对称性和点群对称性的严格约束，在三维空间中只有14种互不等价的类型
- 基元具有极大的自由度：原子种类（元素周期表100+种）、原子数目（1到几千）、原子在基元内的排列方式
- 每一种不同的基元 + 布拉菲格子组合就是一种不同的晶体结构

## 1.3 十四种布拉菲格子的分类

14种布拉菲格子分属7大晶系，按对称性由低到高：

| 晶系 | 点阵参数约束 | 布拉菲格子 | 符号 |
|------|-------------|-----------|------|
| 三斜（Triclinic） | $\mathbf{a}_1 \neq \mathbf{a}_2 \neq \mathbf{a}_3$，$\alpha \neq \beta \neq \gamma$ | 简单三斜 | P |
| 单斜（Monoclinic） | $\mathbf{a}_1 \neq \mathbf{a}_2 \neq \mathbf{a}_3$，$\alpha = \gamma = 90^\circ \neq \beta$ | 简单单斜、底心单斜 | P, C |
| 正交（Orthorhombic） | $\mathbf{a}_1 \neq \mathbf{a}_2 \neq \mathbf{a}_3$，$\alpha = \beta = \gamma = 90^\circ$ | 简单、底心、体心、面心 | P, C, I, F |
| 四角（Tetragonal） | $\mathbf{a}_1 = \mathbf{a}_2 \neq \mathbf{a}_3$，$\alpha = \beta = \gamma = 90^\circ$ | 简单四角、体心四角 | P, I |
| 三角（Rhombohedral） | $\mathbf{a}_1 = \mathbf{a}_2 = \mathbf{a}_3$，$\alpha = \beta = \gamma \neq 90^\circ$ | 三角 | R |
| 六角（Hexagonal） | $\mathbf{a}_1 = \mathbf{a}_2 \neq \mathbf{a}_3$，$\alpha = \beta = 90^\circ, \gamma = 120^\circ$ | 六角 | P |
| 立方（Cubic） | $\mathbf{a}_1 = \mathbf{a}_2 = \mathbf{a}_3$，$\alpha = \beta = \gamma = 90^\circ$ | 简单立方、体心立方、面心立方 | P, I, F |

**记忆要点**：三斜（1种）→ 单斜（2种）→ 正交（4种）→ 四角（2种）→ 三角（1种）→ 六角（1种）→ 立方（3种）= 14种。共计：1+2+4+2+1+1+3 = 14。

## 1.4 典型晶体结构

### 面心立方（FCC）相关结构

| 晶体 | 布拉菲格子 | 基元 | 配位数 | 键合方式 |
|------|-----------|------|--------|---------|
| Cu（铜） | FCC | 1个Cu原子 @ (0,0,0) | 12 | 金属键 |
| NaCl | FCC | Na$^+$ @ (0,0,0) + Cl$^-$ @ $(\frac{1}{2},0,0)$ | 6 | 离子键 |
| 金刚石（C） | FCC | 2个C @ (0,0,0) + $(\frac{1}{4},\frac{1}{4},\frac{1}{4})$ | 4 | 共价键（sp$^3$） |
| 闪锌矿（ZnS） | FCC | Zn @ (0,0,0) + S @ $(\frac{1}{4},\frac{1}{4},\frac{1}{4})$ | 4 | 共价键+离子性 |

### CSI 结构（CsCl结构）

布拉菲格子：简单立方（SC）。基元：Cs$^+$ @ (0,0,0) + Cl$^-$ @ $(\frac{1}{2},\frac{1}{2},\frac{1}{2})$。配位数：8。

### 六角密堆积（HCP）

不是布拉菲格子！HCP是**简单六角格子**附上两个原子的基元：原子1 @ (0,0,0) + 原子2 @ $(\frac{2}{3},\frac{1}{3},\frac{1}{2})$。配位数：12，堆积比 0.74，与 FCC 相同。

**Cu的配位数是12（FCC最密堆积），金刚石的配位数是4**：Cu是单原子FCC基元→最密堆积→12个最近邻。金刚石虽也是FCC格子，但基元含两个C原子→每个C与周围4个C形成sp$^3$四面体共价键→配位数4。

## 1.5 晶体对称性

### 对称操作

晶体中的对称操作包括：
- **平移对称**：$\mathbf{r} \to \mathbf{r} + \mathbf{R}_n$（布拉菲格子的定义性对称性）
- **旋转对称**：绕某轴旋转 $2\pi/n$（$n = 1, 2, 3, 4, 6$）
- **反演**：$\mathbf{r} \to -\mathbf{r}$
- **镜面反射**
- **旋转-反演**
- **螺旋轴**和**滑移面**（包含平移分量的对称操作，空间群特有）

### 晶体学限制定理

只有 $n = 1, 2, 3, 4, 6$ 重旋转轴能与平移对称性共存。5重和 $n \geq 7$ 重旋转对称性在周期晶体中不可能存在。

**证明思路**：设布拉菲格子中存在 $n$ 重旋转轴。绕该轴旋转 $2\pi/n$ 后，晶格必须映射到自身。由平移对称性要求，旋转前后最近邻格矢的矢量和必须是格矢，这给出 $\cos(2\pi/n)$ 必须为半整数，即 $2\cos(2\pi/n) \in \mathbb{Z}$。仅有的解为 $n = 1, 2, 3, 4, 6$。

### 点群与空间群

**32个晶体学点群**：晶体中所有不包含平移的对称操作构成的群（由旋转、反演、镜面组合而成）。

**230个空间群**：包含平移对称操作（螺旋轴、滑移面）的全部对称操作构成的群。

## 1.6 晶面指数（米勒指数）与晶向指数

### 米勒指数 $(hkl)$

晶面与基矢 $\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$ 的截距分别为 $s_1, s_2, s_3$（以基矢长度为单位的数值）。则米勒指数为：

$$(hkl) \propto \left(\frac{1}{s_1}, \frac{1}{s_2}, \frac{1}{s_3}\right)$$

取最简整数比。$(hkl)$ 表示一组平行的晶面族。

**非互质指数的含义**：$(200)$ 与 $(100)$ 取向相同，但晶面间距为 $(100)$ 的一半。在衍射分析中，$(200)$ 对应 $n=2$ 的 $(100)$ 反射。同一晶面族 $\{hkl\}$ 包括所有对称等价的晶面。

### 晶面间距（立方晶系）

$$d_{hkl} = \frac{a}{\sqrt{h^2 + k^2 + l^2}}$$

一般公式：$d_{hkl} = \frac{2\pi}{|\mathbf{K}_{hkl}|}$，其中 $\mathbf{K}_{hkl} = h\mathbf{b}_1 + k\mathbf{b}_2 + l\mathbf{b}_3$ 为倒格矢。

### 晶向指数 $[uvw]$

晶向 $\mathbf{R} = u\mathbf{a}_1 + v\mathbf{a}_2 + w\mathbf{a}_3$ 的方向用 $[uvw]$ 表示，取最简整数比。$\langle uvw \rangle$ 表示所有对称等价的晶向族。

### 晶列夹角（立方晶系）

$$\cos\phi = \frac{u_1u_2 + v_1v_2 + w_1w_2}{\sqrt{u_1^2+v_1^2+w_1^2}\sqrt{u_2^2+v_2^2+w_2^2}}$$

## 1.7 倒格子

### 定义

对于正格子基矢 $\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$，倒格子基矢定义为：

$$\mathbf{b}_1 = 2\pi\frac{\mathbf{a}_2 \times \mathbf{a}_3}{\Omega},\quad \mathbf{b}_2 = 2\pi\frac{\mathbf{a}_3 \times \mathbf{a}_1}{\Omega},\quad \mathbf{b}_3 = 2\pi\frac{\mathbf{a}_1 \times \mathbf{a}_2}{\Omega}$$

其中 $\Omega = \mathbf{a}_1 \cdot (\mathbf{a}_2 \times \mathbf{a}_3)$ 为正格子原胞体积。

**正交关系**：$\mathbf{a}_i \cdot \mathbf{b}_j = 2\pi\delta_{ij}$

**倒格矢**：$\mathbf{K}_h = h_1\mathbf{b}_1 + h_2\mathbf{b}_2 + h_3\mathbf{b}_3$（$h_1,h_2,h_3 \in \mathbb{Z}$）

### 倒格子的重要性质

1. **倒格子体积**：$\Omega^* = \mathbf{b}_1 \cdot (\mathbf{b}_2 \times \mathbf{b}_3) = (2\pi)^3/\Omega$

2. **倒格矢与晶面间距**：倒格矢 $\mathbf{K}_{hkl}$ 垂直于米勒指数为 $(hkl)$ 的晶面族，且 $|\mathbf{K}_{hkl}| = 2\pi/d_{hkl}$

3. **FCC的倒格子是BCC**，**BCC的倒格子是FCC**（重要！）

4. **波函数的倒空间周期性**：$\psi_{\mathbf{k}+\mathbf{K}_h}(\mathbf{r}) = \psi_{\mathbf{k}}(\mathbf{r})$——相差一个倒格矢的波矢描述等价的物理状态

### 为何引入倒格子

晶体中的波动现象（电子波、X射线、中子波、格波）在倒格子空间描述最为自然：
- 布拉格衍射条件：$\Delta\mathbf{k} = \mathbf{K}_h$
- 布洛赫定理：波函数由 $\mathbf{k}$ 量子数标记，且 $\mathbf{k}$ 空间具有倒格子周期性
- 能带结构、费米面、声子色散均定义在倒格子空间

## 1.8 第一布里渊区

### 定义

第一布里渊区 = 倒格子的 Wigner-Seitz 原胞。构造方法：从原点出发作所有倒格矢的垂直平分面，包含原点的最小封闭区域。

### 性质

1. 体积等于倒格子原胞体积：$\Omega_{BZ} = (2\pi)^3/\Omega$
2. 具有正格子点群的全部对称性
3. 任意波矢 $\mathbf{k}$ 可以通过加减倒格矢约化到第一布里渊区内（**约化波矢**）
4. **边界满足布拉格反射条件**：$2\mathbf{k} \cdot \mathbf{K}_h = K_h^2$
5. 布里渊区边界上的高对称点（$\Gamma, X, K, L$ 等）是能带极值点和带隙位置

### 常见晶格的第一布里渊区

- **SC**：边长为 $2\pi/a$ 的立方体
- **FCC**：截角八面体（14面体）
- **BCC**：正十二面体
- **六角**：六棱柱

## 1.9 X射线衍射

### 布拉格定律

$$2d_{hkl}\sin\theta = n\lambda$$

衍射条件：入射波矢 $\mathbf{k}$ 与散射波矢 $\mathbf{k}'$ 之差等于倒格矢：$\mathbf{k}' - \mathbf{k} = \mathbf{K}_h$（**劳厄方程**）。

### 多晶衍射

多晶样品中晶粒取向随机分布。只有那些恰好满足布拉格条件的晶面才产生衍射。对于立方晶系：

$$\sin^2\theta = \frac{\lambda^2}{4a^2}(h^2 + k^2 + l^2)$$

衍射峰的序列（从小到大）由允许的 $(h^2+k^2+l^2)$ 值决定：
- SC：1, 2, 3, 4, 5, 6, 8, 9, …（全部正整数，除7,15等）
- BCC：2, 4, 6, 8, 10, 12, …（$h+k+l$ 为偶数）
- FCC：3, 4, 8, 11, 12, 16, …（$h,k,l$ 全奇或全偶）

**晶面指数越大，衍射角越大**：因为 $d_{hkl} \propto 1/\sqrt{h^2+k^2+l^2}$，$d$ 越小 $\to \sin\theta$ 越大。

### 原子散射因子与结构因子

**原子散射因子** $f$：一个原子对X射线的散射能力，取决于原子中的电子分布。$f$ 随 $\sin\theta/\lambda$ 增大而减小。

**结构因子** $S_{\mathbf{K}}$：一个基元内所有原子的散射波在 $\mathbf{K}$ 方向的相干叠加：

$$S_{\mathbf{K}} = \sum_j f_j e^{-i\mathbf{K}\cdot\mathbf{r}_j}$$

衍射强度 $I \propto |S_{\mathbf{K}}|^2$。结构因子为零时，对应衍射峰系统消光。例如：
- BCC：$h+k+l$ 为奇数时消光
- FCC：$h,k,l$ 奇偶混合时消光

# 第二章 晶体的结合

## 2.1 结合类型与特点

| 结合类型 | 结合力来源 | 方向性 | 饱和性 | 键能（eV/原子） | 典型晶体 |
|---------|-----------|--------|--------|----------------|---------|
| 离子键 | 正负离子间静电库仑力 | 无 | 无 | 5–10 | NaCl, CsCl, MgO |
| 共价键 | 共用电子对（自旋反平行） | 有 | 有 | 4–8 | 金刚石, Si, Ge, GaAs |
| 金属键 | 共有化电子与离子实库仑力 | 无 | 无 | 1–4 | Na, Cu, Al, Fe |
| 分子键（范德瓦尔斯） | 瞬时偶极矩相互作用 | 无 | 无 | 0.02–0.3 | Ar, CH$_4$, 有机晶体 |
| 氢键 | H与两个电负性大的原子结合 | 有 | — | 0.1–0.5 | 冰, HF, 蛋白质, DNA |

**实际晶体往往兼有几种结合形式**。例如石墨层内为共价键（sp$^2$杂化），层间为范德瓦尔斯键。GaAs以共价键为主，但也有离子键成分。

## 2.2 离子键

### 基本特征

正负离子间通过静电库仑力结合。NaCl中Na（3s$^1$）失去一个电子变为Na$^+$，Cl（3p$^5$）获得一个电子变为Cl$^-$，均达到满壳层的稳定电子组态。

**无方向性和饱和性**：库仑力是各向同性的长程力，每个离子尽可能多地与反号离子近邻排列（受空间几何约束）。

### 离子晶体的势能

总势能 = 库仑吸引 + 近距排斥（电子云重叠 + 泡利原理）：

$$U_{\text{tot}} = -N\frac{\alpha e^2}{4\pi\varepsilon_0 r} + N\frac{B}{r^n}$$

其中：
- $\alpha$ 为**马德隆常数**（纯几何因子）：$\alpha = \sum_{j\neq i} \frac{\pm 1}{r_{ij}/r}$（对NaCl结构 $\alpha = 1.7476$，CsCl结构 $\alpha = 1.7627$）
- $B$ 和 $n$ 为排斥势参数（$n \sim 8\text{--}10$）

平衡条件 $\mathrm{d}U/\mathrm{d}r|_{r_0} = 0$ 给出：

$$B = \frac{\alpha e^2 r_0^{n-1}}{4\pi\varepsilon_0 n}$$

平衡结合能（每对离子）：

$$U_0 = -\frac{\alpha e^2}{4\pi\varepsilon_0 r_0}\left(1 - \frac{1}{n}\right)$$

### 离子晶体性质

高强度、高硬度、高熔点（库仑力强）；导电性差（无自由电子，离子迁移需高温）；热膨胀系数小；对红外光透明，但对远红外有强烈吸收（离子振动的光学支）。

## 2.3 共价键

### 基本特征

相邻原子各出一个电子，形成自旋反平行的共用电子对——电子同时归属于两个原子。**方向性**来源于价电子轨道的空间取向（如sp$^3$杂化指向正四面体的四个顶点）；**饱和性**来源于每个原子只能提供有限数量的未配对电子形成共价键。

### 金刚石中的 sp$^3$ 杂化

C原子基态：$1s^2 2s^2 2p^2$，只有2个未配对p电子。形成晶体时，一个2s电子被激发到2p（$2s^1 2p^3$），然后2s和3个2p轨道杂化形成4个等价的sp$^3$杂化轨道，指向正四面体的四个顶点。每个C与4个近邻C形成4个$\sigma$键→键角109.5°→金刚石结构。

### 共价晶体性质

高硬度、高熔点；导电性差（电子被局域在共价键中，无自由载流子）；半导体Si/Ge在有限温度下有一定导电性（热激发跨越窄禁带）。

## 2.4 金属键

### 基本特征

原子外层的价电子脱离原子，形成在整个晶体中自由运动的"共有化电子"（电子气），浸没在带正电的离子实背景中。结合力来自共有化电子与离子实之间的库仑吸引。

**无方向性、无饱和性**（金属可以任意形变而不破坏结合）。

### 金属晶体性质

良好的导电性和导热性（共有化电子可自由输运电荷和能量）；良好的延展性和可加工性（金属键非方向性→原子层之间可以滑移而不破坏结合）；金属光泽（自由电子响应电磁波）。

## 2.5 范德瓦尔斯键

### 物理起源

中性原子或分子之间瞬间电荷涨落产生的感应偶极矩-偶极矩相互作用。总是吸引性。随距离衰减很快（$\propto 1/r^6$）。

**三种类型**：
1. **取向力（Keesom）**：极性分子之间永久偶极矩相互作用
2. **诱导力（Debye）**：极性分子诱导非极性分子的偶极矩
3. **色散力（London）**：瞬时偶极矩-感应偶极矩（量子力学起源，普遍存在）

### 分子晶体性质

结合能低 → 晶体质软、熔点低、易升华。典型的惰性气体晶体（Ne, Ar, Kr, Xe）和有机分子晶体。

## 2.6 原子的电负性

**密立根（Mulliken）定义**：电负性 $= 0.18\,(I + Y)$，其中 $I$ 为电离能，$Y$ 为电子亲和能。因子0.18使Li的电负性恰好为1.0。

**电负性与键合方式的关系**：
- 电负性差 $\Delta\chi \gg 1.7$ → 离子键为主（如NaCl: $\Delta\chi \approx 2.1$）
- $\Delta\chi \approx 0$ → 纯共价键（同种原子，如H$_2$, Si）
- 中间情况 → 极性共价键（兼具共价和离子成分，如GaAs）
- 电负性低的金属元素 → 金属键

## 2.7 结合力与结合能

### 结合力的一般特征

所有晶体结合力有两个共同特征（与具体键合类型无关）：
- **远距离表现为吸引**（$r$ 大时势能为负，斜率 $\mathrm{d}U/\mathrm{d}r > 0$）
- **近距离表现为强排斥**（$r$ 小时势能急剧上升，电子云重叠 + 泡利不相容原理）

平衡位置对应势能最小值：$\mathrm{d}U/\mathrm{d}r|_{r_0} = 0$。

### 两体势模型

晶体总势能写为原子对势能之和：

$$U_{\text{tot}} = \frac{1}{2}\sum_{i\neq j} u(r_{ij})$$

常用两体势形式：$u(r) = -\frac{A}{r^m} + \frac{B}{r^n}$（$n > m$）

- 第一项：吸引势（离子键 $m=1$，范德瓦尔斯 $m=6$）
- 第二项：排斥势（$n \sim 8\text{--}12$，远比 $m$ 大，排斥是短程的）

### 结合能与体弹性模量

**结合能** $E_b = E_N - E_0$（$E_0$ 为晶体基态能量，$E_N$ 为 $N$ 个自由原子能量）。

**体弹性模量** $B = V\frac{\mathrm{d}^2U}{\mathrm{d}V^2}$ 由势能在平衡位置附近的二阶导数决定。$B$ 越大，晶体越"硬"，越难压缩。这一关系直接连接了微观势函数与宏观力学性质。

# 第三章 晶格振动动力学

## 3.1 晶格动力学的基本框架

### 从静态到动态

前两章研究的是静态晶格（原子固定在平衡位置）。实际上原子在平衡位置附近做热振动——这就是晶格动力学的研究对象。晶格振动决定了晶体的热学性质（比热、热膨胀、热传导）以及与光、中子、电子的相互作用。

### 绝热近似

处理晶格振动时采用绝热近似：电子运动远快于离子运动 → 在考虑离子运动时，电子始终处于基态（对应瞬时离子位形的电子基态） → 离子感受到的有效势 $U(\mathbf{R}_1, \mathbf{R}_2, \ldots)$ 由电子基态能量决定。

### 简谐近似

将总势能在平衡位置附近展开到二阶（位移 $\mathbf{u}$ 的二次项）：

$$U = U_0 + \frac{1}{2}\sum_{l\alpha, l'\beta} \Phi_{\alpha\beta}(l,l') u_{l\alpha} u_{l'\beta} + \cdots$$

其中 $\Phi_{\alpha\beta}(l,l') = \frac{\partial^2 U}{\partial u_{l\alpha}\partial u_{l'\beta}}|_0$ 为**力常数矩阵**（一阶项在平衡位置为零）。

运动方程（经典）：

$$M_l \mathrm{d}ot{u}_{l\alpha} = -\sum_{l'\beta} \Phi_{\alpha\beta}(l,l') u_{l'\beta}$$

在简谐近似下，运动方程是**线性的** → 不同格波模式**相互独立、没有相互作用**。

## 3.2 一维单原子链

### 模型

$N$ 个质量为 $m$ 的相同原子排列成一维链，间距为 $a$（晶格常数），仅最近邻有相互作用（力常数 $c$）。周期性边界条件。

### 运动方程与色散关系

第 $n$ 个原子的运动方程：

$$m\mathrm{d}ot{u}_n = c(u_{n+1} + u_{n-1} - 2u_n)$$

设试探解 $u_n = A e^{i(qna - \omega t)}$（$q = 2\pi m/Na$），代入得：

$$-m\omega^2 = c(e^{iqa} + e^{-iqa} - 2) = -2c(1 - \cos qa) = -4c\sin^2\frac{qa}{2}$$

$$\boxed{\omega(q) = 2\sqrt{\frac{c}{m}}\left|\sin\frac{qa}{2}\right|}$$

### 色散关系分析

- $q \to 0$（长波极限）：$\omega \approx \sqrt{c/m}\,a|q| \propto |q|$，格波→弹性波（声波），$v_s = a\sqrt{c/m}$
- $q = \pm\pi/a$（布里渊区边界）：$\omega_{\max} = 2\sqrt{c/m}$，$\mathrm{d}\omega/\mathrm{d}q = 0$（群速度为零→驻波）
- $\omega$ 是 $q$ 的周期函数：$\omega(q + 2\pi/a) = \omega(q)$
- $q$ 取值限制在第一布里渊区 $(-\pi/a, \pi/a]$，共有 $N$ 个独立 $q$ 值

**相速度**：$v_p = \omega/q$。**群速度**：$v_g = \mathrm{d}\omega/\mathrm{d}q = a\sqrt{c/m}\cos(qa/2)$。在布里渊区边界 $v_g = 0$→驻波→布拉格反射。

### 周期性边界条件

Born-von Karman条件：将有限链首尾相连成环。$\psi(x+Na) = \psi(x) \Rightarrow e^{iqNa} = 1$。

$q = \frac{2\pi}{Na}m$（$m = 0, \pm1, \pm2, \ldots, \pm(N/2)$）。独立 $q$ 值有 $N$ 个（等于原胞数）。

**无限大晶体的极限**（$N \to \infty$）：$q$ 从分立谱变为连续谱，$\Delta q = 2\pi/Na \to 0$。求和 $\sum_q \to \frac{Na}{2\pi}\int_{-\pi/a}^{\pi/a}\mathrm{d}q$。

## 3.3 一维双原子链

### 模型

交替排列的两种原子：质量 $m$（轻）和 $M$（重，$m < M$），原子间距 $a$，原胞长度 $2a$。力常数 $c$。

### 色散关系

运动方程组，设解 $u_{2n} = A e^{i(q\cdot 2na - \omega t)}$, $u_{2n+1} = B e^{i(q\cdot(2n+1)a - \omega t)}$，代入得矩阵方程，令行列式为零：

$$\boxed{\omega^2_{\pm} = c\left(\frac{1}{m} + \frac{1}{M}\right) \pm c\sqrt{\left(\frac{1}{m} + \frac{1}{M}\right)^2 - \frac{4\sin^2 qa}{mM}}}$$

### 声学支 $\omega_-$ 与光学支 $\omega_+$ 的特点

| 特性 | 声学支 | 光学支 |
|------|--------|--------|
| $q \to 0$ 频率 | $\omega_- \to 0$ | $\omega_+ \to \sqrt{2c(1/m + 1/M)}$ |
| $q \to 0$ 原子运动 | **同向**（质心运动） | **反向**（相对运动） |
| $q = \pi/2a$ | $\omega_- = \sqrt{2c/M}$（轻原子静止） | $\omega_+ = \sqrt{2c/m}$（重原子静止） |
| 物理本质 | 连续平移对称性的Goldstone模 | 有能隙的光学激发模 |
| 频率范围 | $0 \to \sqrt{2c/M}$ | $\sqrt{2c/m} \to \sqrt{2c(1/m+1/M)}$ |

**声学支 $q\to0$ 时 $\omega\to0$**：长波声学模→原胞质心整体平移→原子间相对距离不变→恢复力零→频率零。本质是平移对称性导致的无能隙激发（Goldstone模）。

**光学支 $q\to0$ 时 $\omega$ 取最大值**（对 $m<M$）：$q=0$ 时所有原胞振动同相，原胞内两原子反向运动→相对位移最大→恢复力最强→频率最高。

### 频率禁带

在 $\sqrt{2c/M} < \omega < \sqrt{2c/m}$ 区间内无格波传播（频率禁带）。禁带宽度 $\propto |1/\sqrt{m} - 1/\sqrt{M}|$，$m$ 和 $M$ 越接近，禁带越窄。$m=M$ 时（单原子链极限）禁带消失。

## 3.4 三维晶格的格波

### 模式计数

$N$ 个原胞，每原胞 $n$ 个原子，三维晶体：
- 总自由度数：$3nN$
- **声学支**：3支（每空间方向1支）→ $3N$ 个声学声子模式
- **光学支**：$3(n-1)$ 支 → $3(n-1)N$ 个光学声子模式
- 每支格波的独立 $q$ 值数：$N$

例：$n=1$（简单晶格）：仅3支声学波。$n=2$（NaCl、金刚石）：3声学+3光学。

### 动力学矩阵

三维情况下，运动方程：

$$\omega^2(\mathbf{q}) \mathbf{e}(\mathbf{q}) = \mathbf{D}(\mathbf{q}) \mathbf{e}(\mathbf{q})$$

其中 $\mathbf{D}(\mathbf{q})$ 为 $3n \times 3n$ 的**动力学矩阵**：

$$D_{\alpha\beta}(\mathbf{q}) = \frac{1}{\sqrt{M_\kappa M_{\kappa'}}}\sum_{\mathbf{R}} \Phi_{\alpha\beta}(\mathbf{R}) e^{-i\mathbf{q}\cdot\mathbf{R}}$$

本征值 $\omega_j^2(\mathbf{q})$（$j=1,\ldots,3n$）给出 $3n$ 支色散关系；本征矢 $\mathbf{e}_j(\mathbf{q})$ 给出极化方向。

## 3.5 声子

### 定义与性质

**声子**：格波能量的量子化单元。频率为 $\omega_j(\mathbf{q})$ 的格波模式，其能量为 $(n_{j\mathbf{q}} + \frac{1}{2})\hbar\omega_j(\mathbf{q})$。$n_{j\mathbf{q}} = 0,1,2,\ldots$ 为该模式的**声子数**。

声子的基本属性：
- 能量：$\hbar\omega$，准动量：$\hbar\mathbf{q}$
- **玻色子**：自旋为零，服从玻色-爱因斯坦统计
- **化学势 $\mu = 0$**：声子数不守恒——格波可以任意产生和湮灭声子
- 声子是**准粒子**（quasiparticle），不是真实的基本粒子——它是晶格集体激发的量子描述

### 声子统计

声子数由玻色-爱因斯坦分布给出：

$$\langle n(\mathbf{q})\rangle = \frac{1}{e^{\hbar\omega(\mathbf{q})/k_BT} - 1}$$

- $k_BT \gg \hbar\omega$（高温）：$\langle n \rangle \approx k_BT/\hbar\omega \propto T$（经典极限）
- $k_BT \ll \hbar\omega$（低温）：$\langle n \rangle \approx e^{-\hbar\omega/k_BT} \ll 1$（量子冻结）

### 温度一定时声学支声子数 >> 光学支声子数

声学支 $\omega_A$ 可从零开始，$\hbar\omega_A \ll k_BT$ 时 $\langle n_A\rangle \gg 1$。光学支 $\omega_O$ 存在最低频率阈值且一般较高，$\hbar\omega_O \sim k_BT$ 甚至 $\hbar\omega_O > k_BT$，$\langle n_O \rangle$ 远小于声学支。

## 3.6 晶格比热

### 经典理论（杜隆-珀蒂定律）

能均分定理：每个谐振自由度平均能量 $k_BT$（动能 $\frac{1}{2}k_BT$ + 势能 $\frac{1}{2}k_BT$）。$N$ 个原子（$3N$ 个自由度）→ $U = 3Nk_BT$ → $C_V = 3Nk_B = 3R$（与温度无关的实验常数）。

仅在高温（$T \gg \Theta_D$）下正确。**低温下完全失效**：实验给出 $C_V \to 0$（$T \to 0$）。

### 爱因斯坦模型（1907）

**假设**：所有 $3N$ 个谐振子以**同一频率** $\omega_E$ 独立振动。

配分函数和能量：

$$U = 3N\frac{\hbar\omega_E}{2} + \frac{3N\hbar\omega_E}{e^{\hbar\omega_E/k_BT} - 1}$$

**爱因斯坦热容公式**：

$$\boxed{C_V = 3Nk_B\left(\frac{\Theta_E}{T}\right)^2\frac{e^{\Theta_E/T}}{(e^{\Theta_E/T} - 1)^2},\quad \Theta_E = \frac{\hbar\omega_E}{k_B}}$$

- $T \gg \Theta_E$：$C_V \to 3Nk_B$（经典极限，恢复杜隆-珀蒂）
- $T \ll \Theta_E$：$C_V \propto e^{-\Theta_E/T}$（指数衰减 → 定性正确：$T \to 0$ 时 $C_V \to 0$）

**成功**：首次用量子论给出 $C_V \to 0$。**失败**：低温下 $C_V \propto e^{-\Theta/T}$ 比实验的 $T^3$ 衰减太快。原因：单一频率假设过于简单，忽略了可被激发的低频模式。

### 德拜模型（1912）

**假设**：
1. 固体视为连续弹性介质，格波为弹性波：$\omega = v_s q$（$v_s$ 为声速，与 $q$ 无关）
2. 频率分布由弹性波态密度决定：$g(\omega) \propto \omega^2$
3. 引入**德拜截止频率** $\omega_D$ 使总模式数为 $3N$：$\int_0^{\omega_D} g(\omega)\mathrm{d}\omega = 3N$
4. 用同一个平均声速 $v_s$ 处理1支纵波和2支横波

态密度：$g(\omega) = \frac{3V}{2\pi^2 v_s^3}\omega^2$

德拜温度：$\Theta_D = \hbar\omega_D/k_B$（金属典型值 100–400 K，金刚石 $\sim$ 2000 K）

**德拜热容公式**：

$$C_V = 9Nk_B\left(\frac{T}{\Theta_D}\right)^3\int_0^{\Theta_D/T}\frac{x^4 e^x}{(e^x - 1)^2}\mathrm{d} x$$

**低温极限**（$T \ll \Theta_D$）：上面积分 → $\int_0^\infty \frac{x^4 e^x}{(e^x-1)^2}\mathrm{d} x = \frac{4\pi^4}{15}$

$$\boxed{C_V = \frac{12\pi^4}{5}Nk_B\left(\frac{T}{\Theta_D}\right)^3 \propto T^3 \quad\text{（德拜 $T^3$ 定律）}}$$

### 爱因斯坦 vs 德拜：为什么德拜更好？

| 比较点 | 爱因斯坦 | 德拜 |
|--------|---------|------|
| 模型 | 单一频率 $\omega_E$ | 频率连续分布 $0 \to \omega_D$ |
| 低温 $C_V$ | $\propto e^{-\Theta_E/T}$（指数） | $\propto T^3$（幂律） |
| 低频声学模 | 没有（人为引入能隙 $\hbar\omega_E$） | 正确包含（$\omega \propto q$，无能隙） |
| 与实验比较 | 定性对，定量偏差大 | 低温与实验符合很好 |

**根本原因**：真实晶体存在无能隙的声学支格波（$\omega \to 0$ 当 $q \to 0$）。低温下只有 $\hbar\omega \lesssim k_BT$ 的模式被激发——德拜正确描述了这些长波声学模，爱因斯坦则人为引入能隙，导致低温下几乎所有模式都被"冻结"。

### 声子态密度（phonon DOS）

$$g(\omega) = \frac{V}{(2\pi)^3}\sum_j \int_{S_\omega} \frac{\mathrm{d} S}{|\nabla_{\mathbf{q}}\omega_j(\mathbf{q})|}$$

态密度的**范霍夫奇点**（van Hove singularities）：在 $\nabla_{\mathbf{q}}\omega = 0$ 的位置（布里渊区高对称点），$g(\omega)$ 出现奇异性（一维发散、二维跳跃、三维斜率不连续）。这些奇点在实验上（如拉曼光谱、非弹性中子散射）可直接观测。

## 3.7 简谐近似与非简谐效应

### 简谐近似的局限性

简谐近似下：
- 势能 $U(x) \propto x^2$，关于平衡位置完全对称
- 不同模式间无耦合 → 声子间无碰撞 → 理想声子气体
- **无法解释热膨胀**：势对称 → 平均位置始终在平衡位置，与温度无关
- **无法解释热阻（有限热导率）**：声子间无散射 → 热导率无限大
- **无法使声子气体达到热平衡**：需要声子-声子相互作用来重新分配能量

### 非简谐效应的物理后果

计入势能展开的三阶及以上项（$U(x) \propto x^2 + gx^3 + hx^4 + \cdots$）：

**1. 热膨胀**：真实原子间势是不对称的——排斥陡峭、吸引平缓。温度升高 → 振幅增大 → 由于非对称性，原子平均间距向势能上升较缓的方向偏移 → **宏观热膨胀**。

热膨胀系数：$\alpha = \frac{1}{L}\frac{\mathrm{d} L}{\mathrm{d} T} = \frac{\gamma C_V}{3BV}$（Grüneisen关系），其中 $\gamma = -\frac{\mathrm{d}\ln\omega}{\mathrm{d}\ln V}$ 为Grüneisen参数。

**2. 热传导（热阻）**：非简谐项引入声子-声子相互作用。关键过程：
- **正常过程（N过程）**：$\mathbf{q}_1 + \mathbf{q}_2 = \mathbf{q}_3$（准动量守恒），不直接产生热阻
- **倒逆过程（Umklapp过程）**：$\mathbf{q}_1 + \mathbf{q}_2 = \mathbf{q}_3 + \mathbf{K}_h$（准动量大幅改变），**产生热阻的主要机制**

Umklapp过程需要参与的声子波矢足够大（$\gtrsim \frac{1}{2}|\mathbf{K}_h|$）。低温下（$T \ll \Theta_D$）大波矢声子数急剧减少 → Umklapp被冻结 → 热导率随降温急剧增大，在 $T \sim \Theta_D/10$ 到 $\Theta_D/50$ 之间出现峰值。

**3. 声子寿命**：非简谐相互作用使声子具有有限寿命 $\tau$。声子平均自由程 $l = v_s\tau$，热导率 $\kappa = \frac{1}{3}C_V v_s l$。

## 3.8 光子与声子的非弹性散射

### 测量色散关系的实验方法

能量和动量守恒：

$$\hbar\omega_s = \hbar\omega_i \pm \hbar\omega(\mathbf{q}),\quad \hbar\mathbf{k}_s = \hbar\mathbf{k}_i \pm \hbar\mathbf{q} + \hbar\mathbf{K}_h$$

测量入射光和散射光的频率差 $\Delta\omega$ 和波矢差 $\Delta\mathbf{k}$ → 得到一个 $(\omega, \mathbf{q})$ 数据点 → 扫描散射角覆盖所有 $\mathbf{q}$ → 完整的 $\omega(\mathbf{q})$。

### 三种主要技术

| 技术 | 探测对象 | 波矢范围 | 能量范围 |
|------|---------|---------|---------|
| 拉曼散射（Raman） | 光学声子（$q \approx 0$） | $\approx 0$ | 10–500 meV |
| 布里渊散射（Brillouin） | 声学声子（$q \approx 0$） | $\approx 0$ | $\mu$eV–meV |
| 非弹性中子散射（INS） | 所有声子 | 全布里渊区 | 0–200 meV |
| 非弹性X射线散射（IXS） | 所有声子 | 全布里渊区 | meV–eV |

拉曼散射测 $\omega_O(q=0)$（$\Gamma$ 点光学声子频率），布里渊散射测声速（长波声学声子），INS/IXS可测全布里渊区的完整色散关系。

### 离子晶体中光-格波耦合的特殊性

离子晶体中主要讨论**长光学波**与电磁波的相互作用。原因：

1. **声学波不直接耦合光**：长波声学模→正负离子同向运动→不产生振荡电偶极矩
2. **长光学波产生振荡偶极矩**：$q \approx 0$ 时原胞内正负离子反向振动→振荡电偶极矩→与同频率光波电磁场共振耦合
3. **波矢匹配**：可见光/红外光波矢 $k \sim 0$，只能与 $q \approx 0$ 的长波格波满足波矢守恒。短波（大 $q$）格波与光波波矢差 $10^5$ 量级，无法直接耦合
4. **极化激元**：光子-长光学声子强耦合→色散反交叉→混合准粒子——**极化激元（polariton）**。在 $\omega_{TO}$ 和 $\omega_{LO}$ 之间形成剩余射线带（Reststrahlen band），此频率区间内光被100%反射

# 第四章 金属自由电子理论

## 4.1 自由电子气体模型（Sommerfeld模型）

### 模型的基本假设

1. **自由电子近似**：金属中的传导电子（价电子）可在金属内完全自由运动。忽略电子与离子实之间的相互作用（除了将电子限制在金属内部）
2. **独立电子近似**：忽略电子与电子之间的库仑相互作用
3. **凝胶模型（Jellium Model）**：将离子实对电子的作用简化为均匀的正电荷背景，维持系统的电中性。电子在均匀正电荷背景中自由运动

在这三个近似下，金属中的电子体系简化为**无相互作用的费米子气体**，被限制在宏观体积 $V = L^3$ 内。

### 单电子量子力学描述

单电子薛定谔方程（无限深势阱 + 周期性边界条件）：

$$\psi_{\mathbf{k}}(\mathbf{r}) = \frac{1}{\sqrt{V}}e^{i\mathbf{k}\cdot\mathbf{r}},\quad \varepsilon(\mathbf{k}) = \frac{\hbar^2 k^2}{2m}$$

波矢量子化：$\mathbf{k} = \frac{2\pi}{L}(n_x, n_y, n_z)$（$n_x,n_y,n_z \in \mathbb{Z}$）。$\mathbf{k}$ 空间体积 $\Delta k = (2\pi/L)^3$，单位 $\mathbf{k}$ 空间 $\mathbf{k}$ 态数为 $V/(2\pi)^3$。每个 $\mathbf{k}$ 态可容纳自旋相反的两个电子。

### 自由电子气体模型 vs 能带理论

自由电子模型 = 能带理论的最简化极限（$U(\mathbf{r}) = \text{const}$）。自由电子气模型适用于简单金属（碱金属、贵金属）的宽泛性质（电导、热导、比热），能带理论在此基础上加入周期势场，才能解释为什么有些材料是导体、有些是绝缘体。

## 4.2 基态（T = 0 K）：费米球

### 基态的构造

$T = 0\,\text{K}$ 时，$N$ 个无相互作用电子按照泡利不相容原理和能量最低原理填充 $\mathbf{k}$ 空间中的单电子态。电子从 $\mathbf{k}=0$ 开始，依次占据能量最低的状态，直到 $N$ 个电子全部排列完毕。

在 $\mathbf{k}$ 空间中，被占据的最小能量态的集合形成一个球——**费米球**（Fermi sphere）。

### 费米球的参数

由自旋简并度和 $\mathbf{k}$ 空间态密度：

$$N = 2\cdot\frac{V}{(2\pi)^3}\cdot\frac{4\pi}{3}k_F^3 \quad\Rightarrow\quad \boxed{k_F = (3\pi^2 n)^{1/3}}$$

其中 $n = N/V$ 为电子数密度。

典型金属的 $n$ 和 $k_F$：
- Na（BCC, $a=4.23\,\text{Å}$, 1个价电子/原子）：$n = 2.65 \times 10^{28}\,\text{m}^{-3}$，$k_F = 0.92\,\text{Å}^{-1}$
- Cu（FCC, $a=3.61\,\text{Å}$, 1个价电子/原子）：$n = 8.47 \times 10^{28}\,\text{m}^{-3}$，$k_F = 1.36\,\text{Å}^{-1}$
- Al（FCC, $a=4.05\,\text{Å}$, 3个价电子/原子）：$n = 18.1 \times 10^{28}\,\text{m}^{-3}$，$k_F = 1.75\,\text{Å}^{-1}$

**费米能量**：$\boxed{\varepsilon_F = \frac{\hbar^2 k_F^2}{2m} = \frac{\hbar^2}{2m}(3\pi^2 n)^{2/3}}$

**费米速度**：$v_F = \frac{\hbar k_F}{m} \sim 10^6\,\text{m/s}$（约光速的 1/300）

**费米温度**：$\boxed{T_F = \frac{\varepsilon_F}{k_B} \sim 10^4\text{--}10^5\,\text{K}}$

室温（$T \sim 300\,\text{K} \ll T_F$）下电子气体处于**强简并**状态——费米-狄拉克统计与经典玻尔兹曼统计差别巨大。

### 基态能量

$$U_0 = \sum_{|\mathbf{k}| \leq k_F} 2\cdot\frac{\hbar^2 k^2}{2m} = \frac{V}{5\pi^2}\frac{\hbar^2 k_F^5}{2m} = \frac{3}{5}N\varepsilon_F$$

基态（$T=0$）电子的平均能量 $\bar{E} = \frac{3}{5}\varepsilon_F$——不为零！尽管温度为零，电子由于泡利不相容原理被迫占据高动能态。

### 电子密度越大→平均动能越大

$k_F \propto n^{1/3}$，$\bar{E} \propto k_F^2 \propto n^{2/3}$。电子密度越大，必须塞进固定 $\mathbf{k}$ 空间体积的电子就越多，必须占据更高 $k$（更高能量）的态——纯量子效应。

## 4.3 费米-狄拉克分布与热学性质

### 费米-狄拉克分布函数

$T \neq 0$ 时，能量为 $\varepsilon$ 的单电子态被占据的概率：

$$f(\varepsilon) = \frac{1}{e^{(\varepsilon - \mu)/k_BT} + 1}$$

$T = 0$ 时：$f = 1$（$\varepsilon < \mu$），$f = 0$（$\varepsilon > \mu$）——阶跃函数，$\mu(T=0) = \varepsilon_F$。

$T > 0$ 时：分布函数在 $\varepsilon_F$ 附近 $\sim k_BT$ 范围内从 1 连续下降到 0。热激发的图像：深埋在费米球内部（$\varepsilon \ll \varepsilon_F$）的电子，其上方的能态被占据，无法跃迁（泡利阻塞）；只有费米面附近 $\sim k_BT$ 范围内的电子上方有空态，可以被热激发。

### 化学势的温度依赖性

$$\mu(T) = \varepsilon_F\left[1 - \frac{\pi^2}{12}\left(\frac{k_BT}{\varepsilon_F}\right)^2 + \cdots\right]$$

$\mu$ 随温度升高而略微降低（$\propto T^2$ 的微小修正）。三维情况下该修正很小，只有 $T \sim T_F$ 时才显著。

### 电子态密度

三维自由电子的态密度（含自旋简并）：

$$\boxed{g(\varepsilon) = \frac{V}{2\pi^2}\left(\frac{2m}{\hbar^2}\right)^{3/2}\sqrt{\varepsilon}}$$

关键特点：$g(\varepsilon) \propto \sqrt{\varepsilon}$。费米面处态密度：

$$g(\varepsilon_F) = \frac{3}{2}\frac{N}{\varepsilon_F}$$

### 电子比热

**经典理论的失败**：能均分定理 → 每个电子 $\frac{3}{2}k_B$ → $C_V = \frac{3}{2}Nk_B$。实验值远小于此（$\sim 100$ 倍）。

**Sommerfeld理论的正确结果**：

只有费米面附近 $\sim k_BT$ 范围内的电子被激发。可被激发电子数 $\approx g(\varepsilon_F)\cdot k_BT$，每个被激发电子获得能量 $\approx k_BT$ → $U_{\text{exc}} \approx g(\varepsilon_F)(k_BT)^2$。

$$C_V = \frac{\partial U}{\partial T} \approx 2g(\varepsilon_F)k_B^2 T = \frac{\pi^2}{2}\left(\frac{Nk_B}{\varepsilon_F}\right)k_B T$$

$$\boxed{C_V = \frac{\pi^2}{2}Nk_B\frac{T}{T_F} = \gamma T}$$

$\gamma$（Sommerfeld系数）与态密度 $g(\varepsilon_F)$ 成正比，是实验上探测费米面附近态密度的重要窗口。

### 低温下金属的总比热

$$C_V = \gamma T + \beta T^3$$

$\gamma T$：电子贡献。$\beta T^3$：晶格振动贡献（德拜模型）。

极低温（$T \ll \Theta_D$）下，电子贡献 $C_e \propto T$ 超过晶格贡献 $C_l \propto T^3$（因为 $T^3$ 比 $T$ 衰减更快）。$C_V/T = \gamma + \beta T^2$，实验上通过低温比热测量拟合 $\gamma$ 可得到费米面态密度。

## 4.4 电子输运性质

### 弛豫时间近似

电子在实际晶体中受到偏离完美周期性的各种不完整性的散射（杂质、缺陷、晶格振动）。用$\tau$（弛豫时间）唯象描述散射：在 $\mathrm{d} t$ 时间内电子受到碰撞的几率为 $\mathrm{d} t/\tau$。$\tau$ 是统计平均量——连续两次碰撞间的平均自由时间。

### 漂移方程

在弛豫时间近似下，电子在外场中的平均运动方程：

$$m^*\left(\frac{\mathrm{d}\mathbf{v}_d}{\mathrm{d} t} + \frac{\mathbf{v}_d}{\tau}\right) = -e(\mathbf{E} + \mathbf{v}_d \times \mathbf{B})$$

$\mathbf{v}_d$ 为漂移速度（叠加在随机热运动上的定向平均速度），$m^*$ 为有效质量（自由电子气中 $m^* = m_e$）。

### 稳恒电场下的电导率

$\mathrm{d}\mathbf{v}_d/\mathrm{d} t = 0$（稳态），$\mathbf{B} = 0$：

$$\mathbf{v}_d = -\frac{e\tau}{m}\mathbf{E} = -\mu\mathbf{E},\quad \mu = \frac{e\tau}{m}\text{（迁移率）}$$

电流密度 $\mathbf{j} = -ne\mathbf{v}_d$ → 欧姆定律：

$$\mathbf{j} = \sigma\mathbf{E},\quad \boxed{\sigma = \frac{ne^2\tau}{m}}$$

电导率 $\sigma$ 与载流子浓度 $n$ 和弛豫时间 $\tau$ 成正比。$\tau$ 由散射机制决定：低温下杂质散射主导（$\tau \approx$ 常数），高温下声子散射主导（$\tau \propto 1/T$）→ 电阻率 $\rho \propto T$。

### 散射来源

理想晶体中布洛赫电子不受散射（电阻为零）。实际金属的电阻来自对完美周期势的偏离：
- **晶格缺陷**：空位、位错、层错、晶界
- **杂质原子**：替位式或间隙式杂质 → 残余电阻（$T \to 0$ 时不消失）
- **声子散射**：晶格振动 → 声学支声子是室温电阻的主要来源
- **电子-电子散射**：仅在极低温和特殊情况下重要
- **表面/界面散射**：当样品尺寸 < 平均自由程时显著

### 为什么只有费米面附近电子对输运有贡献

电场使整个费米球在 $\mathbf{k}$ 空间平移 $\Delta\mathbf{k} = -e\mathbf{E}\tau/\hbar$。原有费米球与新位置费米球交叠部分的电子对电流无贡献（$\pm\mathbf{k}$ 方向对称抵消）。净电流仅来自费米面附近未被补偿的电子——数量上约占总电子数的 $\sim k_BT/\varepsilon_F$。

## 4.5 维德曼-弗兰兹定律

金属的热导率与电导率之比与温度成正比：

$$\boxed{\frac{\kappa}{\sigma} = LT,\quad L = \frac{\pi^2}{3}\left(\frac{k_B}{e}\right)^2 = 2.44 \times 10^{-8}\,\text{W}\cdot\Omega/\text{K}^2}$$

$L$ 为**洛伦兹数**，在自由电子模型下为普适常数。

物理基础：电子既传导电流，也传导热量。两种输运都由费米面附近电子完成，且受同样的散射机制限制（$\sigma \propto \tau$，$\kappa \propto \tau$），因此比值与散射细节无关而仅取决于统计性质。

## 4.6 功函数与接触电势差

### 功函数

电子从金属内部逸出到真空所需的最小能量：

$$W = E_{\text{vac}} - E_F$$

$E_{\text{vac}}$ 为真空中静止电子的能量。$W$ 对金属表面状态敏感，典型值 2–5 eV。

### 电子热发射（Richardson-Dushman公式）

可逃逸电子需 $E \geq E_F+W$（远在费米能级以上 $\gg k_BT$），费米分布退化为玻尔兹曼分布。电流密度：

$$j = AT^2 e^{-W/k_BT},\quad A = \frac{4\pi mek_B^2}{h^3} = 1.2 \times 10^6\,\text{A/m}^2\text{K}^2$$

### 接触电势差

两种不同金属（功函数 $W_1 \neq W_2$）接触时，电子从费米能级高的金属流向费米能级低的金属，两侧分别带正电和负电，形成接触电势差。平衡时两侧费米能级齐平：

$$V_{\text{contact}} = \frac{W_2 - W_1}{e}$$

## 4.7 霍尔效应

在垂直于电流方向的磁场中，载流导体在横向（同时垂直于电流和磁场）产生电场——**霍尔效应**。

对自由电子气体：
- 霍尔系数 $R_H = \frac{E_y}{j_x B_z} = -\frac{1}{ne}$
- 霍尔电阻率 $\rho_{xy} = \frac{E_y}{j_x} = -\frac{B}{ne}$

由 $R_H$ 的符号可判断载流子类型（电子：负；空穴：正），由其大小可测载流子浓度。实际金属中 $R_H$ 常与自由电子预测有符号和大小偏差——这需要用能带理论（有效质量、空穴）来正确解释。

### 霍尔效应的微观图像

磁场使电子受洛伦兹力偏转，在样品边缘累积电荷，产生横向霍尔电场。稳态时霍尔电场力与洛伦兹力平衡 → $E_y = -v_x B_z$。正常霍尔效应来自洛伦兹力。此外还有反常霍尔效应（磁性材料中，与自旋-轨道耦合相关）、量子霍尔效应（二维电子气，强磁场，低温）和分数霍尔效应。

# 第五章 固体的能带理论

## 5.1 单电子近似——能带理论的起点

### 问题的提出

自由电子气体模型成功地解释了金属的许多性质，但它不能解释：
- 为什么有些材料是导体，而有些是绝缘体？
- 为什么电子有效质量与自由电子质量不同？
- 半导体的存在及其性质

能带理论通过在自由电子模型的基础上恢复离子实的周期性排列，解释了这一切。

### 三个基本近似

晶体中包含 $\sim 10^{23}$ 个相互作用的电子和原子核。严格求解这一多体问题是不可能的。通过以下三个近似，将其简化为可处理的形式：

**（1）绝热近似（Born-Oppenheimer近似）**

$M_{\text{ion}} \gg m_e$ → 离子运动远慢于电子运动。在研究电子时，可认为离子固定在某瞬时位置上。多粒子体系简化为**在固定离子构型下运动的相互作用多电子体系**。

物理图像：离子"重而慢"，电子"轻而快"。电子可以瞬时跟上离子的任何运动（始终处于对应离子位形的电子基态）。

**（2）单电子近似（Hartree-Fock近似/平均场近似）**

多电子问题仍然太复杂。将每个电子看作在离子实产生的势场**和所有其他电子产生的平均场**中独立运动。将相互作用的多电子体系简化为**无相互作用的单电子在有效势场中运动**。

数学上：多电子波函数近似为单电子波函数的Slater行列式。通过变分原理得到Hartree-Fock方程。

**（3）周期场近似**

离子实在晶体中周期排列 → 单电子感受到的有效势也具有晶格周期性：

$$U(\mathbf{r} + \mathbf{R}_n) = U(\mathbf{r})$$

最终：相互作用的多电子+多离子体系 → 周期势场中运动的**无相互作用单电子**问题。由此建立的电子能谱理论即为**能带理论**。

### 与自由电子气体模型的对比

| | 自由电子气体模型 | 能带理论 |
|------|----------------|---------|
| 哈密顿量 | $\hat{H} = -\frac{\hbar^2}{2m}\nabla^2$ | $\hat{H} = -\frac{\hbar^2}{2m}\nabla^2 + U(\mathbf{r})$ |
| 本征态 | 平面波 | 布洛赫波 |
| 本征值 | $E = \hbar^2k^2/2m$ | $E = E_n(\mathbf{k})$（能带结构） |
| 适用范围 | 简单金属的定性描述 | 所有晶态固体 |

两者都是基于无相互作用的单电子模型，服从同样的费米-狄拉克统计。区别在于单电子感受到的势场不同。

## 5.2 布洛赫定理

### 定理陈述

在周期势场 $U(\mathbf{r}+\mathbf{R}_n) = U(\mathbf{r})$ 中运动的单电子，其定态薛定谔方程的本征函数（布洛赫波函数）具有如下形式：

$$\boxed{\psi_{n\mathbf{k}}(\mathbf{r}) = e^{i\mathbf{k}\cdot\mathbf{r}} u_{n\mathbf{k}}(\mathbf{r}),\quad u_{n\mathbf{k}}(\mathbf{r}+\mathbf{R}_n) = u_{n\mathbf{k}}(\mathbf{r})}$$

- $e^{i\mathbf{k}\cdot\mathbf{r}}$：平面波因子（描述波的整体传播特性）
- $u_{n\mathbf{k}}(\mathbf{r})$：周期调制因子（与晶格同周期，描述波在原子尺度的振荡）
- $n$：能带指标
- $\mathbf{k}$：波矢（第一布里渊区内取值）

### 物理意义

布洛赫定理是晶体平移对称性的必然结果——平移算符 $\hat{T}_{\mathbf{R}}$ 与哈密顿量对易，它们必有共同本征函数。$\hbar\mathbf{k}$ 是平移算符的本征值（**晶体动量/准动量**），不是真实的动量（布洛赫波不是动量算符的本征态）。

### 布洛赫定理的推论

1. **波函数和能量在倒格子空间具有周期性**：
   $$\psi_{n,\mathbf{k}+\mathbf{K}_h}(\mathbf{r}) = \psi_{n\mathbf{k}}(\mathbf{r}),\quad E_n(\mathbf{k}+\mathbf{K}_h) = E_n(\mathbf{k})$$
   $\mathbf{k}$ 和 $\mathbf{k}+\mathbf{K}_h$ 是等价的物理状态 → $\mathbf{k}$ 取值可限制在第一布里渊区。

2. **周期性边界条件→$\mathbf{k}$ 取分立值**：每个布里渊区内 $\mathbf{k}$ 可取 $N$ 个值（$N$ 为晶体原胞数）。

3. **每个能带容纳 $2N$ 个电子**（$N$ 个 $\mathbf{k}$ 值 × 自旋简并度 2）。

## 5.3 能带和带隙的形成

### 从自由电子到近自由电子

自由电子的能谱：$E(\mathbf{k}) = \hbar^2k^2/2m$——抛物线，连续，无禁带。

加入弱的周期势场 $U(\mathbf{r})$：
- **远离布里渊区边界**：$U(\mathbf{r})$ 的影响很小 → 电子行为近似自由电子 → 能量接近抛物线
- **接近布里渊区边界**：周期势场引起强烈的布拉格反射 → 入射波和反射波干涉 → 形成**驻波**
  - 较低能量的"成键"驻波（波腹在离子位置附近→势能低）
  - 较高能量的"反键"驻波（波腹在离子之间→势能高）
- 两个驻波的能量差即为**禁带宽度（带隙）**：$E_g = 2|U(\mathbf{K}_h)|$

### 能带形成的物理图像总结

孤立原子：分立的原子能级（如1s, 2s, 2p, …）。

原子靠近形成晶体：
- 原子波函数发生交叠 → 能级展宽为**能带**
- 原子间距越小、波函数交叠越多 → 能带越宽
- 不同原子能级展宽的能带可能发生重叠（能带杂化）
- 能带之间的能量间隙对应**禁带（带隙）**

## 5.4 近自由电子近似

### 基本思想与适用条件

周期势场非常弱 → 自由电子情况可取为零级近似 → 周期势作为微扰。

适用于：简单金属的价电子（碱金属、贵金属等），离子实对价电子的作用较弱。

### 一维情形的具体计算

非简并微扰（$\mathbf{k}$ 远离布里渊区边界）：

$$E(\mathbf{k}) \approx \frac{\hbar^2k^2}{2m} + U_0$$

简并微扰（$\mathbf{k} \approx \frac{1}{2}K$，布里渊区边界附近）：

$$E_{\pm} = \frac{1}{2}(E_0(\mathbf{k}) + E_0(\mathbf{k}-K)) \pm \sqrt{\frac{1}{4}(E_0(\mathbf{k}) - E_0(\mathbf{k}-K))^2 + |U_K|^2}$$

在布里渊区边界（$\mathbf{k} = \frac{1}{2}K$）：
$$E_{\pm} = \frac{\hbar^2K^2}{8m} + U_0 \pm |U_K|$$

**带隙 $E_g = 2|U_K|$**。带的标记：$|U_K|$ = $|U_1|$（第一带隙），$|U_2|$（第二带隙），以此类推。

### 三维的推广——能带图

三维情况下 $\mathbf{k}$ 空间的不同方向的色散关系不同。通常在布里渊区的高对称方向画 $E(\mathbf{k})$ 来表现能带结构。$\Gamma$（$\mathbf{k}=0$）, $X$, $L$, $K$ 等是高对称点。

## 5.5 平面波方法

### 基本思想

将布洛赫波函数按平面波完备基展开：

$$\psi_{\mathbf{k}}(\mathbf{r}) = \sum_{\mathbf{K}} C_{\mathbf{k}-\mathbf{K}} e^{i(\mathbf{k}-\mathbf{K})\cdot\mathbf{r}}$$

将周期势也展开为傅里叶级数：$U(\mathbf{r}) = \sum_{\mathbf{K}} U_{\mathbf{K}} e^{i\mathbf{K}\cdot\mathbf{r}}$。代入薛定谔方程得到**中心方程**（平面波表象中的代数本征值方程）：

$$(\varepsilon_{\mathbf{k}-\mathbf{K}}^0 - E)C_{\mathbf{k}-\mathbf{K}} + \sum_{\mathbf{K}'} U_{\mathbf{K}'-\mathbf{K}} C_{\mathbf{k}-\mathbf{K}'} = 0$$

实际计算时截断平面波数目（取有限个 $\mathbf{K}$），数值对角化矩阵得到能带 $E(\mathbf{k})$。

### 平面波方法的本质

近自由电子近似 = 平面波方法的微扰论极限（只取少数几个平面波）。完整的平面波方法适用于任意强度的周期势（取足够多的平面波）。

## 5.6 紧束缚方法（LCAO方法）

### 基本思想

电子在原子附近主要受该原子势场作用 → 晶体中电子波函数与孤立原子波函数相似 → 用孤立原子波函数的线性组合（LCAO：Linear Combination of Atomic Orbitals）来构造布洛赫波函数：

$$\psi_{n\mathbf{k}}(\mathbf{r}) = \frac{1}{\sqrt{N}}\sum_{\mathbf{R}} e^{i\mathbf{k}\cdot\mathbf{R}}\phi_n(\mathbf{r}-\mathbf{R})$$

其中 $\phi_n$ 为孤立原子的第 $n$ 个原子轨道。

### 能带表达式

$$\boxed{E_n(\mathbf{k}) = \varepsilon_n - J_0 - \sum_{\mathbf{R} \neq 0} J(\mathbf{R}) e^{i\mathbf{k}\cdot\mathbf{R}}}$$

各项物理含义：
- $\varepsilon_n$：孤立原子的第 $n$ 个能级
- $J_0$：其他所有原子对该原子处电子能级的整体下移（晶体场劈裂）
- $J(\mathbf{R}) = \int \phi_n^*(\mathbf{r})[U(\mathbf{r}) - V_{\text{atom}}(\mathbf{r})]\phi_n(\mathbf{r}-\mathbf{R})\mathrm{d}\mathbf{r}$：**迁移积分/跳跃积分**——电子从位置 $\mathbf{R}$ 的原子跳到原点原子的概率幅

### 迁移积分的物理含义

- $J$ 越大 → 电子巡游性越强 → 能带越宽
- 带宽 $\propto$ 最近邻 $J$ 值 $\times$ 配位数 $z$
- 孤立原子能级 → 能带的展宽：$\Delta E \approx 2zJ$
- 原子间距越小 → 波函数交叠越多 → $J$ 越大 → 带宽越大

### 紧束缚方法 vs 近自由电子近似 vs 平面波方法

| | 近自由电子近似 | 紧束缚方法 | 平面波方法 |
|------|--------------|-----------|----------|
| 出发点 | 自由电子+弱周期势微扰 | 孤立原子+近邻原子微扰 | 数值精确解 |
| 基函数 | 少数平面波 | 原子轨道（LCAO） | 大量平面波 |
| 适用对象 | 弱周期势（简单金属价电子） | 紧束缚电子（$d$/$f$电子、绝缘体） | 任意体系 |
| 能带本质 | 周期势对平面波的布拉格散射 | 原子能级因波函数交叠而展宽 | 薛定谔方程的直接对角化 |

### BCC晶格 $s$ 态电子的紧束缚能带

对BCC结构仅考虑最近邻（8个），$J(\mathbf{R}) = J_1$（对最近邻）：

$$\boxed{E(\mathbf{k}) = \varepsilon_s - J_0 - 8J_1\cos\frac{k_x a}{2}\cos\frac{k_y a}{2}\cos\frac{k_z a}{2}}$$

沿 $[100]$ 方向（$k_y = k_z = 0$）：$E(k_x) = \varepsilon_s - J_0 - 8J_1\cos\frac{k_x a}{2}$。带宽 = $16J_1$。

带底：$(0,0,0)$处 $E_{\min} = \varepsilon_s - J_0 - 8J_1$。
带顶：$(\pm 2\pi/a, 0, 0)$或等价点处 $E_{\max} = \varepsilon_s - J_0 + 8J_1$。

### 能带杂化

当不同原子能级（如 $s$ 和 $p$）展开的能带有能量重叠时，不能仅用单一原子轨道展开 → 需用多个原子轨道的线性组合 → 能带杂化。这是能带结构复杂性的主要来源。

## 5.7 半经典模型

### 核心思想

内部周期势场对电子的作用 → 量子力学处理（能带理论，全部封装在 $E_n(\mathbf{k})$ 中）。外场（电场、磁场）对电子的作用 → 经典牛顿力学处理。电子被看作波包，具有确定的位置 $\mathbf{r}$、波矢 $\mathbf{k}$ 和能带指标 $n$。

### 三条基本规则

**（1）能带指标 $n$ 是运动常数**：外场下电子始终在同一能带中运动，不发生带间跃迁。

**（2）电子平均速度**：

$$\boxed{\mathbf{v}_n(\mathbf{k}) = \frac{1}{\hbar}\nabla_{\mathbf{k}}E_n(\mathbf{k})}$$

布洛赫态不是速度的本征态，$\mathbf{v}_n$ 是量子力学平均速度，完全由能带结构决定。

**（3）波矢运动方程**：

$$\boxed{\hbar\frac{\mathrm{d}\mathbf{k}}{\mathrm{d} t} = -e\left[\mathbf{E} + \mathbf{v}_n(\mathbf{k})\times\mathbf{B}\right]}$$

形式与自由电子的牛顿方程 $\hbar\mathrm{d}\mathbf{k}/\mathrm{d} t = \mathbf{F}$ 完全相同（只要用晶体动量 $\hbar\mathbf{k}$ 代替真实动量）。

### 适用范围

- 外场波长 $\lambda \gg a$（晶格常数）——电子可视为波包
- 电场 $eEa \ll E_g^2/E_F$——避免齐纳隧穿（电子在不同能带间的电场诱导隧穿）
- 外场频率 $\hbar\omega \ll E_g^2/E_F$——避免带间跃迁的光吸收

## 5.8 有效质量

### 定义

在周期势场中，电子在外力 $\mathbf{F}$ 作用下的加速度：

$$\frac{\mathrm{d} v_\alpha}{\mathrm{d} t} = \frac{1}{\hbar}\frac{\mathrm{d}}{\mathrm{d} t}\frac{\partial E}{\partial k_\alpha} = \frac{1}{\hbar}\sum_\beta\frac{\partial^2 E}{\partial k_\alpha\partial k_\beta}\frac{\mathrm{d} k_\beta}{\mathrm{d} t}$$

$$\boxed{\frac{1}{m_{\alpha\beta}^*} = \frac{1}{\hbar^2}\frac{\partial^2 E(\mathbf{k})}{\partial k_\alpha \partial k_\beta}}$$

有效质量是**二阶张量**，一般是各向异性的。在能带极值点（$\Gamma$点等），有效质量可以简化为标量。

### 有效质量的物理意义

$$\mathbf{F}_{\text{ext}} + \mathbf{F}_{\text{lattice}} = m_e\mathbf{a} \quad\longrightarrow\quad \mathbf{F}_{\text{ext}} = m^*\mathbf{a}$$

有效质量将内部周期势场对电子的复杂作用力等效地"吸收"进一个参数 $m^*$，使得电子在外场下的运动方程形式上保持牛顿第二定律的形式。

- $m^* > 0$（能带底部）：外力与加速度同向
- $m^* < 0$（能带顶部）：外力与加速度反向——晶格力超过外力，电子对外力的响应反常
- $|m^*|$ 小 → 能带曲率大 → 电子"轻"（响应灵敏）
- $|m^*|$ 大 → 能带平 → 电子"重"（响应迟钝）

### 负有效质量的物理图像

当电子处于能带顶部时，外电场试图加速电子，但随着 $\mathbf{k}$ 增大（接近布里渊区边界），布拉格反射增强，电子的群速度反而减小。外力和加速度方向相反 → $m^* < 0$。解决这一"反常"的自然方案：用带正电荷、正有效质量的**空穴**替代能带顶部的电子。

## 5.9 空穴

### 定义与性质

满带顶部附近的空状态（少了一个电子）在外场下的集体行为完全等价于一个：
- 带正电荷 $+e$
- 具有正有效质量 $|m^*|$
- 能量向下增加（"空穴能量"）

的假想粒子——**空穴**（hole）。

### 为何引入空穴

满带中有 $2N-1$ 个电子，直接处理这 $2N-1$ 个电子的运动几乎不可能。引入空穴后，只需处理**一个**空穴的运动即可描述整个满带（减一个电子的状态）的电流。这是能带理论中极为精妙的概念简化。

### 空穴的来源

空穴不是真实粒子，而是描述满带中大量电子集体行为的等效准粒子。它的"正电荷"和"正有效质量"是对能带顶部电子负有效质量和负电荷的两次负号抵消的结果。

## 5.10 稳恒电场中 Bloch 电子的运动

### $\mathbf{k}$ 空间运动

$\mathrm{d}\mathbf{k}/\mathrm{d} t = -e\mathbf{E}/\hbar$ → 所有电子以相同速率在 $\mathbf{k}$ 空间移动。到达布里渊区边界时，被布拉格反射到对面边界（等价于 $\mathbf{k} \to \mathbf{k} + \mathbf{K}_h$）。在 $\mathbf{k}$ 空间形成周期性轨道。

### 实空间运动——布洛赫振荡

将 $\mathbf{k}(t)$ 代入 $\mathbf{v}(\mathbf{k})$ → 电子在实空间做周期性振荡运动（**布洛赫振荡**），振荡周期 $\tau_B = h/(eEa)$。在一般电场下 $\tau_B \sim 10^{-5}$ s，远大于实际晶体中的散射弛豫时间（$\sim 10^{-14}$ s），因此在常规固体中难以观测。在超晶格（人工周期远大于自然晶格常数）和光晶格中已实现布洛赫振荡的观测。

### 满带不导电、不满带导电

- **满带**：所有 $\mathbf{k}$ 态的占据是对称的（$\mathbf{k}$ 和 $-\mathbf{k}$ 被等量占据）→ 净电流为零。电场使整个分布移动，但由于布里渊区的周期性，总的占据分布保持不变 → 净电流仍然为零
- **不满带**：电场使占据分布偏移 → 破坏 $\pm\mathbf{k}$ 对称 → 净电流非零 → 导电

## 5.11 稳恒磁场中 Bloch 电子的运动

### $\mathbf{k}$ 空间轨道

由 $\hbar\mathrm{d}\mathbf{k}/\mathrm{d} t = -e\mathbf{v}\times\mathbf{B}$ 和 $\mathbf{v} = \frac{1}{\hbar}\nabla_{\mathbf{k}}E$ 可得：
- $\mathrm{d}\mathbf{k}/\mathrm{d} t \perp \nabla_{\mathbf{k}}E$ → 电子沿等能面运动
- $\mathrm{d}\mathbf{k}/\mathrm{d} t \perp \mathbf{B}$ → 轨道在垂直于 $\mathbf{B}$ 的平面内

→ $\mathbf{k}$ 空间轨道 = 等能面与 $\perp\mathbf{B}$ 平面的交线。

### 实空间轨道

实空间中垂直 $\mathbf{B}$ 平面的轨道 = $\mathbf{k}$ 空间轨道绕磁场轴旋转 $90^\circ$ + 尺度缩放。沿 $\mathbf{B}$ 方向一般做变速运动。

### 朗道轨道与朗道能级

从量子力学角度，电子在垂直于磁场的平面内的回旋运动被**量子化**，形成分立的**朗道能级**（Landau levels）：

- 自由电子：$E_\nu = (\nu + \frac{1}{2})\hbar\omega_c$（等间距），$\omega_c = eB/m_c$ 为回旋频率
- 简并度（单位面积）：$n_B = eB/h$
- 在 $\mathbf{k}$ 空间中：原本准连续分布的 $\mathbf{k}$ 态被"压缩"到一系列**朗道环**上——环截面面积被量子化为 $S_\nu = \frac{2\pi eB}{\hbar}(\nu + \frac{1}{2})$

朗道量子化是所有磁场中电子量子振荡现象的根源。

## 5.12 固体的导电性——能带填充

### 最核心的结论

固体的电学性质取决于**最高占据能带（价带）和最靠近的空带（导带）之间的能量关系和电子填充情况**。

| 类型 | 能带填充 | 禁带宽度 | 导电行为 | 例子 |
|------|---------|---------|---------|------|
| 金属（导体） | 价带部分填充，或价带与导带重叠 | 无禁带 | 良导体，$\rho \uparrow$ 随 $T\uparrow$ | Na, Cu, Al, Ag |
| 绝缘体 | 价带全满，导带全空 | 宽（$E_g > 3$–$5$ eV） | 不导电 | 金刚石（5.5 eV）, SiO$_2$（9 eV） |
| 本征半导体 | 价带全满，导带全空 | 窄（$E_g \sim 0.1$–$2$ eV） | 热激发导电，$\rho \downarrow$ 随 $T\uparrow$ | Si（1.1 eV）, Ge（0.67 eV）, GaAs（1.43 eV） |
| 半金属 | 价带与导带小幅重叠 | 负禁带 | 类金属导电，载流子极少 | Bi, Sb, As, 石墨 |

### 一价金属导电、二价金属可为绝缘体

- 一价金属（Na, K, Cu, Ag, Au）：每个原胞1个价电子。能带容纳 $2N$ 个电子，电子只占一半 → 半满 → 导体
- 二价金属（Mg, Ca, Zn）：每个原胞2个价电子。如果能带不重叠，价带恰好填满 → 可能为绝缘体。但实际中由于能带重叠，这些材料仍为金属
- 二价绝缘体：如果布里渊区边界处的带隙足够大且能带无重叠 → 绝缘体

## 5.13 费米面的重要性

费米面是 $\mathbf{k}$ 空间中 $E = E_F$ 的等能面。几乎所有低能物理过程仅涉及费米面附近的电子：

- **电导/热导**：仅费米面附近电子参与输运
- **电子比热**：$C_V = \gamma T$，$\gamma \propto g(E_F)$
- **泡利顺磁磁化率**：$\chi \propto g(E_F)$
- **德哈斯-范阿尔芬效应**：费米面极端截面积的直接测量
- **超导电性**（BCS理论中）：$T_c \propto \Theta_D e^{-1/g(E_F)V}$

## 5.14 德哈斯-范阿尔芬效应

### 现象

在低温（$T \ll T_F$）和强磁场（$\omega_c\tau \gg 1$）下，金属的磁化率（以及其他物理量如电导率、比热等）随 $1/B$ 呈现出**周期性振荡**。

### 物理机制

磁场的改变使朗道能级在能量空间中移动，朗道环的截面面积 $S_\nu = \frac{2\pi eB}{\hbar}(\nu + \frac{1}{2})$ 随之改变。每当一个朗道环扫过费米面的极端截面时，费米面附近的态密度发生突变 → 磁化率等物理量发生振荡。

### 振荡周期

$$\boxed{\Delta\left(\frac{1}{B}\right) = \frac{2\pi e}{\hbar S_{\text{ext}}}}$$

$S_{\text{ext}}$：费米面在垂直于磁场方向的**极端截面积**。

不同磁场方向给出不同的 $S_{\text{ext}}$ → 通过扫描磁场方向可重构整个费米面的三维形状——这是实验上确定金属费米面结构最直接、最有力的方法。

## 5.15 布洛赫电子在外场中的行为总结

### 外场下的关键点

- 在外场下，Bloch态不再是严格本征态，电子在Bloch态之间迁移（$\mathbf{k}(t)$ 含时）
- 半经典模型适用时，每个瞬间可用Bloch态描述（能带指标守恒）
- 能带结构 $E_n(\mathbf{k})$ 完整决定了电子在外场下的动力学

### 各种外场情况汇总

| 外场 | $\mathbf{k}$ 空间运动 | 实空间运动 | 关键现象 |
|------|---------------------|-----------|---------|
| 稳恒电场 | $\mathrm{d}\mathbf{k}/\mathrm{d} t = -e\mathbf{E}/\hbar$（等速）→ 布里渊区周期循环 | 布洛赫振荡 | 满带不导电/不满带导电 |
| 稳恒磁场 | $\perp\mathbf{B}$ 平面内沿等能面运动 | $\mathbf{k}$ 轨道旋转 $90^\circ$ | 回旋共振，朗道量子化 |
| 电场+磁场 | 复杂轨道 | 霍尔效应 | 正常/反常/量子霍尔效应 |
| 温度梯度 | — | — | 热电效应（Seebeck效应） |

### 在外场作用下Bloch态的性质

在外场下，Bloch态不再是严格的本征态。电子在Bloch态之间连续迁移（$\mathbf{k}$ 随时间变化），但每个时刻电子的状态仍可用Bloch波函数近似描述（半经典近似的核心假设）。能带指标 $n$ 不发生改变——这是半经典模型的第一条规则，本质上是量子绝热定理的一个特例。

## 5.16 能带计算的现代方法简介

实际能带计算的发展超出了本课程范围，但思想基础仍是本章的三个近似和两种方法（平面波法和紧束缚法）的融合。现代方法包括：
- **缀加平面波方法（APW）**
- **正交化平面波方法（OPW）**
- **赝势方法**：用平滑的赝势替代离子实区域的强库仑势，大幅减少需要的平面波数目
- **密度泛函理论（DFT）**：基于Hohenberg-Kohn定理，将相互作用多电子体系的基本性质完全由电子密度决定。Kohn-Sham方程将相互作用电子映射为有效势中的无相互作用电子，是现代第一性原理计算的标准框架

# 附录：核心公式速查

## 晶体结构

$$\mathbf{a}_i \cdot \mathbf{b}_j = 2\pi\delta_{ij}$$

$$d_{hkl} = \frac{2\pi}{|\mathbf{K}_{hkl}|},\quad \mathbf{K}_{hkl} = h\mathbf{b}_1 + k\mathbf{b}_2 + l\mathbf{b}_3$$

立方晶系：$d_{hkl} = \frac{a}{\sqrt{h^2+k^2+l^2}}$

布拉格定律：$2d\sin\theta = n\lambda$

FCC ↔ BCC 互为倒格子

## 晶体的结合

两体势：$u(r) = -\frac{A}{r^m} + \frac{B}{r^n}$

平衡：$\frac{\mathrm{d} u}{\mathrm{d} r}|_{r_0} = 0$，稳定：$\frac{\mathrm{d}^2 u}{\mathrm{d} r^2}|_{r_0} > 0$

离子晶体：$U = -N\frac{\alpha e^2}{4\pi\varepsilon_0 r_0}(1 - \frac{1}{n})$

## 一维单原子链

$$\omega(q) = 2\sqrt{\frac{c}{m}}\left|\sin\frac{qa}{2}\right|,\quad v_g = a\sqrt{\frac{c}{m}}\cos\frac{qa}{2}$$

## 一维双原子链

$$\omega^2_{\pm} = c\left(\frac{1}{m}+\frac{1}{M}\right) \pm c\sqrt{\left(\frac{1}{m}+\frac{1}{M}\right)^2 - \frac{4\sin^2 qa}{mM}}$$

## 德拜模型

低温比热：$C_V = \frac{12\pi^4}{5}Nk_B\left(\frac{T}{\Theta_D}\right)^3$

$$g(\omega) = \frac{3V}{2\pi^2 v_s^3}\omega^2,\quad \Theta_D = \frac{\hbar\omega_D}{k_B}$$

## 自由电子气体

$$k_F = (3\pi^2 n)^{1/3},\quad \varepsilon_F = \frac{\hbar^2 k_F^2}{2m},\quad T_F = \frac{\varepsilon_F}{k_B}$$

$$g(\varepsilon_F) = \frac{3}{2}\frac{N}{\varepsilon_F},\quad U_0 = \frac{3}{5}N\varepsilon_F$$

$$C_V = \frac{\pi^2}{2}Nk_B\frac{T}{T_F} = \gamma T$$

$$\sigma = \frac{ne^2\tau}{m},\quad R_H = -\frac{1}{ne}$$

## 布洛赫定理

$$\psi_{n\mathbf{k}}(\mathbf{r}) = e^{i\mathbf{k}\cdot\mathbf{r}}u_{n\mathbf{k}}(\mathbf{r}),\quad u_{n\mathbf{k}}(\mathbf{r}+\mathbf{R}) = u_{n\mathbf{k}}(\mathbf{r})$$

$$E_n(\mathbf{k}+\mathbf{K}_h) = E_n(\mathbf{k})$$

## 近自由电子近似

带隙：$E_g = 2|U_{\mathbf{K}}|$

## 紧束缚能带

$$E(\mathbf{k}) = \varepsilon_0 - J_0 - \sum_{\mathbf{R}}J(\mathbf{R})e^{i\mathbf{k}\cdot\mathbf{R}}$$

BCC $s$ 带：$E(\mathbf{k}) = \varepsilon_s - J_0 - 8J_1\cos\frac{k_x a}{2}\cos\frac{k_y a}{2}\cos\frac{k_z a}{2}$

## 有效质量

$$\frac{1}{m_{\alpha\beta}^*} = \frac{1}{\hbar^2}\frac{\partial^2 E}{\partial k_\alpha \partial k_\beta}$$

## 半经典运动方程

$$\mathbf{v} = \frac{1}{\hbar}\nabla_{\mathbf{k}}E(\mathbf{k}),\quad \hbar\frac{\mathrm{d}\mathbf{k}}{\mathrm{d} t} = -e\mathbf{E} - e\mathbf{v}\times\mathbf{B}$$

## 德哈斯-范阿尔芬效应

$$\Delta\left(\frac{1}{B}\right) = \frac{2\pi e}{\hbar S_{\text{ext}}}$$

## 朗道能级

$$E_\nu = \left(\nu + \frac{1}{2}\right)\hbar\omega_c,\quad \omega_c = \frac{eB}{m_c},\quad n_B = \frac{eB}{h}$$
