# 第一章 晶体结构

#### 1. 阐述单晶、多晶和非晶的差别。
- **单晶**：原子在整个材料中周期性排列，具有<mark style="background:#fff88f">长程有序</mark>，方向一致，具有各向异性，不存在晶界
- **多晶**：由晶粒组成，每个晶粒内部是有序的，但晶粒之间取向不同，存在晶界
- **非晶**：原子无长程有序，只有短程有序，电子衍射图为弥散的晕环
#### 2. 布拉菲格子中的格点都是等价的吗？
布拉菲格子具有严格的<mark style="background:#fff88f">平移对称性</mark>，其任意格点都可以通过晶格矢量$\mathbf{R}=n_1\mathbf{a}_1+n_2\mathbf{a}_2+n_3\mathbf{a}_3$从原点获得。对于任意两个格点，总存在一个晶格平移矢量将一个格点映射到另一个格点，同时保持整个晶体的结构不变。因此，每个格点的局域环境完全相同。在几何物理上不可区分，故布拉菲格子中所有格点都是等价的。
#### 3. 一个原胞含有几个格点，几个原子？魏格纳-赛兹原胞有什么特点？
原胞的定义是<mark style="background:#fff88f">能够平移操作无重叠铺满整个晶格的最小体积单元</mark>，在周期性平铺时，每个原胞对应一个独立的平移矢量，每个平移矢量对应一个格点，所以一个原胞含有一个格点。<mark style="background:#fff88f">原胞中的原子数为基元中的原子数</mark>。
**Wigner-Seitz Cell构造方法**：
1. 选定一个格点
2. 连到所有最近邻格点
3. 作这些线的垂直平分面
4. 围成区域即为WS原胞
特点：
- 只包含一个格点
- 具有与晶格相同的对称性
- 内部所有点距离该格点最近
- 其倒空间对应为第一布里渊区
#### 4. Cu, NaCl和金刚石晶体结构的相同之处和不同之处。
共同点：它们的布拉菲格子均为面心立方（FCC），都属于立方晶系，具有较高的对称性，并可表示为“布拉菲格子+基元“的形式。
不同点：
- **基元不同**：Cu为单原子基元，NaCl为两个不同的原子组成的基元，金刚石为两个相同原子组成的基元
- **配位数不同**：Cu为12配位（最密堆积），NaCl为6配位（八面体结构），金刚石为4配位（四面体结构）
- **键合方式不同**：Cu为金属键，NaCl为离子键，金刚石为共价键
- 原子排列方式和物理性质也存在差异
#### 5. Cu和金刚石都是面心立方，为什么Cu的配位数是12，而金刚石的配位数是4？
Cu是单原子基元，每个格点都被原子占据，形成最密堆积，因此配位数为12，而金刚石是FCC格子附上两个原子组成的基元，第二个原子位于$(\frac{1}{4},\frac{1}{4},\frac{1}{4})$，只形成四面体的共价键，因此每个原子只有4个最近邻，配位数为4。
#### 6. 晶格点阵与实际晶体有何区别和联系？为什么晶格点阵只有14种，而实际的晶体结构有成千上万？
晶格点阵是由平移对称性定义的抽象几何点集，仅描述晶体的周期性结构，不包含具体原子信息；而实际上晶体结构是在晶格点阵的每个格点上附加一定的基元；三维空间中晶格点阵只有14种，是因为在平移对称性和空间对称性的约束下，只有14种互不等价的布拉菲格子，而实际晶体可以有成千上万种是因为基元具有很大的自由度，包括原子种类，数量和内部自由度。
#### 7. 为什么晶体结构没有5重旋转对称性？
晶体结构中要求具有平移对称性，而旋转对称操作必须与平衡对称性相容。根据晶体学限制定理，只有<mark style="background:#fff88f">1、2、3、4、6重旋转轴</mark>能够与平移对称性共存。5重旋转对称性无法使空间周期性填充，会产生空隙或重叠。
#### 8. 根据晶面指数的定义，$(h_1, h_2, h_3)$ 中 $h_1, h_2, h_3$ 三个数是互质的，为什么文献中有晶面指数为（2，0，0）或（4，2，0）这样的形式？
米勒指数在定义时要求取最简整数比，是为了唯一表示晶面的取向。但实际应用中，如（2，0，0）（4，2，0）表示的是与最简晶面平行但间距不同的晶面，即同一晶面族中不同的成员，它们对应不同的晶面间距。
#### 9. 倒格式描述的是什么空间？为什么要引进倒格式？
倒格子是由一组与实空间晶格矢量正交关系定义的矢量所构成的点阵，用于描述晶体的周期性在波矢空间的表现。因为晶体中的电子、光和声等物理问题本质上是波动性质相关，在k空间中可以更自然地描述周期性、干涉和能带结构。
#### 10. 倒格式中的维格纳-赛兹原胞（即第一布里渊区）有什么特点？
1. 是倒格子的原胞，体积最小且只含有一个倒格点
2. 任意波矢都可以通过加减倒格矢映射到第一布里渊区
3. 具有与实空间晶格相同的对称性
4. 其边界对应布拉格反射条件，是能带开隙的重要区域
5. 所有与电子结构相关的物理量，均定义在第一布里渊区
#### 11. 多晶样品X射线衍射谱中，衍射峰的（角度）位置与晶面指数有什么关系？
在多晶样品的X射线中，衍射峰的角度位置由布拉格定律$2d\sin\theta=n\lambda$，其中晶面间距$d$与晶面指数$(hkl)$有关。对于立方晶系，$d_{hkl}=\frac{a}{h^2+k^2+l^2}$，因此衍射角满足$\sin^2\theta\propto(h^2+k^2+l^2)$。这表示晶<mark style="background:#fff88f">面指数越大，对应的衍射角越大</mark>。不同的$(hkl)$对应不同的衍射峰位置，多晶样品中所有满足条件都会产生衍射峰。

# 第二章 晶体的结合

#### 1. 阐述离子键、共价键、金属键和范德瓦尔斯键的特点，由这几种键构成的晶体有什么基本属性？
1. **离子键**：由正、负离子之间的静电吸引力形成；特点：<mark style="background:#fff88f">无方向性，无饱和性，键能较高，电负性差异大</mark>
2. **共价键**：由相邻原子共享一对或多对电子形成；特点：有方向性、有饱和性、键能高、电负数、性相近的非金属
3. **金属键**：由自由电子气与正离子实之间的静电作用形成；特点：无方向性，无饱和性，电子高度离域，可在晶体中自由移动
4. **范德瓦尔斯键**：由分子或原子间瞬间的偶极矩相互作用；特点：键能弱，存在于所有分子或者、惰性气体原子之间
- **离子晶体**：强度高、硬度熔点高、导电性热膨胀系数小
- **共价晶体**：饱和性和方向性，硬度熔点高，导电性差
- **分子晶体**：结合能低、晶体质软、熔点低
- **金属晶体**：导热性和导电性好
#### 2. 如何理解共价键的饱和性和方向性？
**饱和性**：一个原子只能形成一定数目的共价键。
**方向性**：通常在价电子电荷密度最大的方向形成共价键。
#### 3. 为什么金属比离子晶体、共价晶体易于进行机械加工并且导电、导热性好？
因为金属键的基本特征是<mark style="background:#fff88f">电子的共有化运动</mark>，即在结合成晶体时，原来属于各原子的价电子不再束缚在原子上，而转变为在整个晶体内运动，它的波函数遍布整个晶体。存在共有化电子，所以金属具有良好的导热性和导电性。
#### 4. 阐述晶体中离子实之间相互作用势的一般特点。
在短距离处表现为<mark style="background:#fff88f">强烈的排斥作用</mark>，主要来源于电子云重叠及泡利不相容原理；在中远距离处表现为<mark style="background:#fff88f">吸引作用</mark>，来源于库伦相互作用或范德华力；势能曲线存在一个最小值，对应稳定的平衡距离即晶格常数。势函数具有明显的非对称性（非简谐性），在平衡位置附近可近似为简谐势，实际相互作用具有多体性，不完全是简单的两体势。
#### 5. 什么是原子的电负性？原子间的结合与原子的电负性有什么关系？
电负性是衡量原子得失电子难易的物理量    <mark style="background:#fff88f"> 0.18（原子电离势+电子亲和势）</mark>
1. 当两个成键原子的电负性差值较大时，晶体结合往往采取离子键，由周期表最左端和最右端的元素结合成晶体，主要是离子键
2. 同种原子之间的成键，主要是共价键或金属键，因为原子的电负性一样大
3. 电负性差值小的原子之间成键主要是共价键，向元素周期表中的相邻元素之间形成的主要是共价键，但是也有一定的离子键成份，价电子不仅为两个原子共享，而且还偏向于电负性较大的原子一边

# 第三章 晶格动力学

#### 1. 如何描述一个波？晶体中的格波和连续介质中的弹性波有什么差别？对非晶态固体材料，格波和弹性波的概念是否还适用？为什么？
描述波的本质是求解波动方程，可以得到色散关系。弹性波是连续介质中偏微分方程的解，格波是离散晶格中差分方程的解。
差别：源于周期性边界条件。格波有布里渊区边界和截止频率，且波矢不唯一。

| 对比维度          | **弹性波（连续介质）**           | **格波（晶体）**                                                                                                                      |
| ------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **介质模型**      | 连续、均匀，无内部结构             | 离散、周期性排列的原子点阵                                                                                                                   |
| **空间坐标**      | $X$是连续实数，场量定义在所有点上      | 位置仅定义在离散格点$na$上                                                                                                                 |
| **色散关系**      | **线性**：$\omega=vk$，波速恒定 | **周期性的正弦函数**，在布里渊区边界（$k=\pm\frac{\pi}{a}$）处$\frac{dω}{dk}=0$（群速为零）                                                              |
| **波矢$k$的唯一性** | **唯一**。不同的 kk 对应不同的物理振动 | **不唯一**。因为$e^{ikna}=e^{i(k+G)na}$其中$G=\frac{2\pi}{a}m$为倒格矢），$k$和 $k+G$描述**完全相同的原子位移**。因此必须限制在**第一布里渊区**（$−π/a<k≤π/a−π/a<k≤π/a$） |
| **截止频率**      | 无上限，频率可随 $k$无限增大        | **存在上限**：$ω_{max}=2\sqrt{\frac{\beta}{M}}$。短波长的波无法在晶格中传播（对应原子间距量级的振动）                                                           |
| **物理本源**      | 宏观的应力-应变关系              | 微观的原子间相互作用（简谐近似）                                                                                                                |

**最重要的物理图像补充（长波极限）**：  
当波长$\lambda\gg a$时，即$k\to0$，$\sin(\frac{ka}{2})\approx\frac{ka}{2}$，格波的色散关系退化为$\omega\approx\sqrt{\frac{\beta}{M}}\cdot a\cdot k$。此时波速$v=\sqrt{\frac{\beta}{M}}\cdot a$为常数，**格波完全蜕变为弹性波**。这说明弹性波只是格波在长波长的低频近似。
格波不适用，因为格波的数学建立（Bloch定理、色散关系$\omega(k）$、布里渊区)严格以来晶体的平移对称性。非晶态没有周期性晶格，原子间距不是严格等距的$a$，因此不存在固定的倒格矢$G$，波矢$k$不再是刻画振动模式的好量子数。
弹性波在低频长波下使用，高频失效。

#### 2. 周期性边界条件的物理含义是什么？引入这个条件后导致什么结果？如果晶体是无限大，$q$ 的取值将会怎样？

**物理含义**：周期性边界条件（Born-von Karman边界条件）将有限大小的晶体首尾相连，形成一个没有边界的环形结构。数学表述为：$\psi(x + Na) = \psi(x)$，即第 $N+1$ 个原子的振动状态与第1个原子完全相同。其物理实质是**消除表面效应**——真实晶体中表面原子与体内原子受力环境不同，会产生复杂的表面振动模式，周期性边界条件通过将晶体拓扑等价为一个环（一维）或环面（三维），使所有原子处于完全等价的环境中。

**引入后的结果**：
- 波矢 $q$ 取分立值：$e^{iqNa} = 1 \Rightarrow q = \frac{2\pi}{Na}m$（$m$ 为整数）
- 独立的 $q$ 值数目等于原胞数 $N$，且限制在第一布里渊区 $(-\pi/a, \pi/a]$ 内
- 格波模式数是有限的、可数的：一维单原子链有 $N$ 个独立模式，三维晶体 $N$ 个原胞、每原胞 $n$ 个原子共有 $3nN$ 个模式
- 求和 $\sum_q$ 变为在离散 $q$ 点上求和

**如果晶体无限大**（$N \to \infty$）：$q$ 取值间隔 $\Delta q = 2\pi/Na \to 0$，$q$ 从分立谱变为连续谱。求和 $\sum_q$ 过渡为积分 $\frac{Na}{2\pi}\int_{-\pi/a}^{\pi/a}\mathrm{d}q$。

#### 3. 以一维双原子链为例，阐述晶格振动所产生的格波的性质。解释为什么波矢 $k\to0$ 声学支格波的频率趋近于零，而光学支格波的频率趋近于最大。

**一维双原子链模型**：质量分别为 $m$ 和 $M$（设 $m < M$）的两种原子交替排列，间距 $a$，最近邻力常数 $c$。色散关系：

$$\omega^2_{\pm} = c\left(\frac{1}{m} + \frac{1}{M}\right) \pm c\sqrt{\left(\frac{1}{m} + \frac{1}{M}\right)^2 - \frac{4\sin^2 qa}{mM}}$$

两支格波的性质：

**声学支（$\omega_-$）**：
- 原胞内两个原子**同向运动**（质心运动），相邻原胞间有相位差
- $q \to 0$ 时 $\omega_- \to 0$：长波极限下，原胞内两原子几乎同步运动，整个晶体如同连续介质中的弹性波（声波），恢复力来自原子间相对位移的累积效应
- 频率范围：$0 \le \omega_- \le \sqrt{2c/M}$
- $q = \pi/2a$ 时，轻原子 $m$ 静止，只有重原子 $M$ 振动

**光学支（$\omega_+$）**：
- 原胞内两个原子**反向运动**（相对运动），原胞质心基本不动
- $q \to 0$ 时 $\omega_+ \to \sqrt{2c(1/m + 1/M)}$（有限值，频率最大）：长波极限下相邻原胞的振动相位相同，原胞内正负离子反向振动产生振荡电偶极矩，可与光波电磁场耦合（因此称为"光学支"）
- 频率范围：$\sqrt{2c/M} \le \omega_+ \le \sqrt{2c(1/m + 1/M)}$
- $q = \pi/2a$ 时，重原子 $M$ 静止，只有轻原子 $m$ 振动

**$q = \pi/2a$ 处**两支格波之间存在频率间隙（禁带）：$\sqrt{2c/M} < \omega < \sqrt{2c/m}$，此频率范围内无格波传播。

**$q \to 0$ 声学支 $\omega \to 0$ 的原因**：长波声学模对应原胞质心的整体平移——平移不改变原子间相对距离，恢复力为零，故频率为零。这是Goldstone定理的体现：连续对称性（平移对称性）的自发破缺导致无能隙的Goldstone模。

**$q \to 0$ 光学支 $\omega$ 取最大值的原因**（对 $m < M$）：$q=0$ 时所有原胞振动同相，每个原胞内两原子以最大振幅反向振动——相对位移最大，恢复力最强，频率最高。$q$ 增大时相邻原胞相位不同，两原子的相对振幅减小，频率降低。

#### 4. 什么是简谐近似？简谐近似下不同格波模式间有没有相互作用？

**简谐近似**：将原子间相互作用势 $U(\mathbf{r})$ 在平衡位置附近作泰勒展开，仅保留到二阶项（位移的二次项）：

$$U = U_0 + \frac{1}{2}\sum_{l,l'}\sum_{\alpha,\beta}\frac{\partial^2 U}{\partial u_{l\alpha}\partial u_{l'\beta}}\bigg|_0 u_{l\alpha}u_{l'\beta} + \cdots$$

其中一阶项在平衡位置为零。二阶项对应的恢复力与位移成正比（胡克定律形式），运动方程为线性方程。

**简谐近似下格波模式间没有相互作用**。原因：
- 运动方程是线性的，不同 $q$ 模式之间完全解耦——通过正则变换（引入简正坐标），哈密顿量化为 $3nN$ 个独立谐振子之和
- 声子是严格的自由玻色子，彼此不发生碰撞和能量交换
- 声子气体的热平衡无法通过自身达到——需要借助非简谐效应（声子-声子散射）

**简谐近似的局限性**：无法解释热膨胀（平衡原子间距不随温度变化）、热传导（声子间无散射则热导率无限大）、热阻等现象。这些都需要引入三阶及以上的非简谐项。

#### 5. （1）阐述晶格振动的格波理论中声学支格波和光学支格波的基本特点。（2）由 $N$ 个原胞组成的晶体，每个原胞中有 $n$ 个原子，问：晶格振动中有几支声学波，有几支光学波，每支格波的格波数是多少？

**（1）基本特点**

| | 声学支格波 | 光学支格波 |
|---|---|---|
| $q \to 0$ 频率 | $\omega \to 0$ | $\omega \to$ 有限最大值 |
| 原胞内原子运动 | **同向**（质心运动） | **反向**（相对运动） |
| 长波物理图像 | 连续介质弹性波（声波） | 振荡电偶极矩（可与光耦合） |
| 频率范围 | 从零到截止频率 | 从截止频率到最大值 |
| 群速度（$q \to 0$） | 等于声速（常数） | 趋近于零 |
| 物理本质 | 原胞质心的集体平移 | 原胞内原子的相对振动 |

**更本质的对称性理解**：声学支对应连续平移对称性的Goldstone模——整个晶体的均匀平移不消耗能量，因此 $\omega(q=0)=0$。光学支则是有能隙的激发模式。

**（2）模式计数**

对于 $N$ 个原胞、每原胞 $n$ 个原子的三维晶体：
- 总自由度：$3nN$（$N$ 个原胞 × $n$ 个原子/原胞 × 3个空间方向）
- **声学支数**：$3$ 支（每空间方向1支）→ 共 $3N$ 个声学声子模式
- **光学支数**：$3(n-1)$ 支 → 共 $3(n-1)N$ 个光学声子模式
- 每支格波的格波数（独立 $q$ 值数）：$N$ 个（等于原胞数，$q$ 在第一布里渊区内取 $N$ 个分立值）

特例：$n=1$（简单晶格，如单原子链）只有3支声学波，无光学波。$n=2$（如NaCl、金刚石）有3支声学波 + 3支光学波。

#### 6. 为什么离子晶体中讨论电磁波与格波的相互作用主要讨论长光学波模式，不是短光学波模式也不是声学支格波？

**核心原因**：

1. **声学支格波不与光波耦合**：长波声学模对应原胞质心的整体平移运动，正负离子同步同向位移，不产生振荡电偶极矩，不能与电磁场交换能量。

2. **长光学波产生振荡电偶极矩**：$q \approx 0$ 的长光学波模式下，正负离子反向振动，原胞内出现随时间振荡的电偶极矩 $\mathbf{p}(t) = e^*\mathbf{u}(t)$（$e^*$ 为有效电荷，$\mathbf{u}$ 为相对位移）。该振荡偶极矩可与同频率的光波电磁场发生共振耦合——这正是光被离子晶体强烈吸收和反射的微观机制（红外活性）。

3. **为什么是长波而非短波**：光波的波矢 $k = \omega/c$ 在红外频段（$\omega \sim 10^{13}\,\text{Hz}$）约为 $k \sim 10^3\,\text{cm}^{-1}$，而晶格布里渊区尺寸 $\sim 10^8\,\text{cm}^{-1}$。因此光波矢 $k \ll \pi/a$，只能与 $q \approx 0$ 的长波格波满足波矢匹配条件。短光学波（$q$ 接近布里渊区边界）的波长与光波波长相差 $10^5$ 倍，波矢守恒 $\mathbf{k} = \mathbf{q} \pm \mathbf{K}_h$ 可以通过倒格矢参与的过程（多声子过程或高阶过程）间接耦合，但强度远弱于长波直接耦合。

4. **后果——极化激元**：长光学声子与光子强耦合导致两者的色散曲线在交叉点发生反交叉（avoided crossing），形成混合准粒子——**极化激元（polariton）**，其色散关系在共振频率 $\omega_{TO}$ 和 $\omega_{LO}$ 之间存在禁带（剩余射线带，Reststrahlen band）。

#### 7. 什么是声子？声子满足什么统计规律？为什么声子体系的化学势为零？

**声子的定义**：晶格振动能量的量子化单元。频率为 $\omega_i(\mathbf{q})$ 的格波模式，其能量只能取分立值 $(n + \frac{1}{2})\hbar\omega_i$，$n = 0,1,2,\ldots$ 为该模式的声子数。声子不是真实粒子，而是描述晶格集体激发的**准粒子（quasiparticle）**，准动量为 $\hbar\mathbf{q}$。

**统计规律**：声子是玻色子，服从**玻色-爱因斯坦统计**：

$$\langle n(\mathbf{q})\rangle = \frac{1}{e^{\hbar\omega(\mathbf{q})/k_BT} - 1}$$

**化学势为零的原因**：声子数不守恒。晶格振动可以通过热激发任意产生声子，也可以通过非简谐相互作用湮灭声子。系统没有守恒的声子数约束——系统通过自由调节声子数来最小化自由能，因此化学势 $\mu = 0$。在统计力学的框架下，$\mu = (\partial F/\partial N)_{T,V}$，由于 $N$（声子数）没有物理意义（不需守恒），平衡条件自然给出 $\mu = 0$。

这与光子气体的化学势为零基于同样的原理——都是粒子数不守恒的玻色子体系。

#### 8. （1）温度一定，光学支格波的声子数目多，还是声学支格波的声子数目多？为什么？（2）对同一个振动模式，温度高时声子数目多，还是温度低时声子的数目多？为什么？（3）高温时，频率为 $\omega$ 的格波的声子数目与温度有何关系？

**（1）温度一定时，声学支声子数多于光学支声子数。**

原因：声子数由玻色-爱因斯坦分布决定 $\langle n\rangle = 1/(e^{\hbar\omega/k_BT} - 1)$。
- 声学支 $\omega_A$ 低（可从零开始），$\hbar\omega_A \ll k_BT$ 时 $\langle n_A\rangle \approx k_BT/\hbar\omega_A \gg 1$，大量声子被激发
- 光学支 $\omega_O$ 高（存在最小频率阈值），通常 $\hbar\omega_O \sim k_BT$ 甚至 $\hbar\omega_O > k_BT$，$\langle n_O\rangle$ 显著小于声学支

例如室温下（$k_BT \approx 25\,\text{meV}$），声学支声子频率可低至 $\sim 1\,\text{meV}$，$\langle n\rangle \sim 25$；而光学支频率通常 $\sim 30\text{–}100\,\text{meV}$，$\langle n\rangle < 1$。因此声学支声子在室温下占绝对主导。

**（2）温度高时声子数目多。**

原因：同一个振动模式（$\omega$ 固定），$\langle n\rangle = 1/(e^{\hbar\omega/k_BT} - 1)$ 是 $T$ 的增函数。温度升高 → $k_BT$ 增大 → $e^{\hbar\omega/k_BT}$ 减小 → $\langle n\rangle$ 增大。

物理图像：温度越高，热运动越剧烈，原子振动的振幅越大，对应声子数越多。

**（3）高温时（$k_BT \gg \hbar\omega$）**：

$$\langle n\rangle = \frac{1}{e^{\hbar\omega/k_BT} - 1} \approx \frac{k_BT}{\hbar\omega} \propto T$$

声子数与温度成正比。此时量子效应不重要，每个谐振模式均分能量 $k_BT$（经典能均分定理），声子数 $\langle n\rangle \gg 1$，格波处于经典极限。

#### 9. （1）阐述晶格比热的爱因斯坦模型和德拜模型的物理含义，（2）为什么用德拜模型计算的比热在低温时能给出很好的结果，而用爱因斯坦模型计算的结果比较差？

**（1）物理含义**

**爱因斯坦模型（1907）**：
- 假设晶体中所有 $3N$ 个谐振子以**同一频率** $\omega_E$ 独立振动
- 物理图像：每个原子被固定在平衡位置，以相同频率振动，相邻原子间无耦合
- 配分函数和热容可直接从量子谐振子解析求出
- 特征温度 $\Theta_E = \hbar\omega_E/k$

成功之处：预言了 $T \to 0$ 时 $C_V \to 0$（量子效应导致热容趋于零），突破了经典杜隆-珀蒂定律。首次用量子论解决了低温比热问题。

**德拜模型（1912）**：
- 将固体视为**连续弹性介质**，格波为弹性波（$\omega = v_s q$，$v_s$ 为声速）
- 频率分布由弹性波的态密度决定：$g(\omega) \propto \omega^2$
- 引入**截止频率** $\omega_D$（德拜频率），使总模式数恰好为 $3N$：$\int_0^{\omega_D} g(\omega)\mathrm{d}\omega = 3N$
- 特征温度 $\Theta_D = \hbar\omega_D/k$

**（2）低温下德拜模型优于爱因斯坦模型的原因**

爱因斯坦模型在低温（$T \ll \Theta_E$）下给出 $C_V \propto e^{-\Theta_E/T}$（指数衰减），而德拜模型给出 $C_V \propto T^3$（幂律衰减）。

**根本原因**：真实晶体没有能隙。周期晶格中存在声学支格波，其频率可以从零连续变化到截止频率。低温下 $k_BT$ 很小，只有 $\hbar\omega \lesssim k_BT$ 的低频模式能被激发：
- **德拜模型**正确地包含了 $\omega \to 0$ 的声学模，态密度 $\propto \omega^2$ 在低频端不为零，低温下可被激发的声子数 $\propto T^3$ → $C_V \propto T^3$
- **爱因斯坦模型**假设所有振子同频 $\omega_E$，相当于人为引入一个能隙 $\hbar\omega_E$。低温下 $k_BT \ll \hbar\omega_E$ 时，几乎所有振子都被"冻结"，只有极少数被激发 → $C_V \propto e^{-\Theta_E/T}$（指数衰减太快）

物理本质：爱因斯坦模型忽略了长波声学模——这些模式频率低、无能隙，是低温比热的主要来源。德拜模型通过连续介质近似正确描述了长波声学模的 $\omega \propto q$ 色散和 $\omega^2$ 态密度。

#### 10. 为什么简谐近似下的晶格振动理论解释不了晶体的热膨胀和热阻现象？阐述晶体热膨胀和热阻的来源。

**简谐近似无法解释的原因**：
- 简谐势 $U(x) = \frac{1}{2}cx^2$ 关于平衡位置对称 → 原子振动的平均位置始终等于平衡位置，与振动幅度（温度）无关 → **热膨胀系数为零**
- 简谐近似下不同格波模式解耦，声子之间不发生碰撞 → 声子一旦被激发就永远以相同方向传播（无Umklapp过程） → **热导率无限大（热阻为零）**

**热膨胀的来源——非简谐效应**：真实原子间势能是不对称的（排斥势陡峭、吸引势平缓）。温度升高 → 原子振动振幅增大 → 由于势的非对称性，平均原子间距向势能上升较缓的一侧（吸引侧，即间距增大的方向）偏移 → 宏观上表现为热膨胀。

热膨胀系数 $\alpha = \frac{1}{L}\frac{\mathrm{d}L}{\mathrm{d}T}$ 可通过格林艾森参数 $\gamma = -\frac{\mathrm{d}\ln\omega}{\mathrm{d}\ln V}$ 与晶格振动频率的体积依赖性联系起来。

**热阻的来源——声子-声子散射**：
- **正常过程（N过程）**：声子碰撞满足准动量守恒 $\mathbf{q}_1 + \mathbf{q}_2 = \mathbf{q}_3$，总动量不变，不直接产生热阻（但可重新分配声子分布）
- **倒逆过程（Umklapp过程）**：$\mathbf{q}_1 + \mathbf{q}_2 = \mathbf{q}_3 + \mathbf{K}_h$（$\mathbf{K}_h \neq 0$ 为倒格矢），声子准动量发生大幅度改变（"翻转"），产生热阻
- Umklapp过程需要参与碰撞的声子波矢足够大（$\gtrsim \frac{1}{2}\mathbf{K}_h$），因此**低温下Umklapp过程被冻结**（声子主要分布在 $q \approx 0$ 附近），热导率随温度降低急剧上升（峰值热导率效应）
- 此外，杂质散射、边界散射和同位素散射也是热阻来源

#### 11. 阐述实验上用光子与声子的非弹性散射来测量晶格振动色散关系的基本原理，明确说明实验中测量哪些量，以及如何得出色散关系 $\omega(q)$。

**基本原理**：入射光子（频率 $\omega_i$，波矢 $\mathbf{k}_i$）与晶体中的声子发生非弹性散射，产生或湮灭一个声子，散射光子频率 $\omega_s$ 和波矢 $\mathbf{k}_s$ 发生变化。能量和动量守恒：

$$\hbar\omega_s = \hbar\omega_i \pm \hbar\omega(\mathbf{q}),\quad \hbar\mathbf{k}_s = \hbar\mathbf{k}_i \pm \hbar\mathbf{q} + \hbar\mathbf{K}_h$$

$+$ 号对应产生一个声子（Stokes过程），$-$ 号对应湮灭一个声子（anti-Stokes过程），$\mathbf{K}_h$ 为倒格矢（大角度散射时可能出现）。

**实验中测量的量**：
- 入射光频率 $\omega_i$（已知的单色光源，通常为激光）
- 散射光频率 $\omega_s$ → 频率差 $\Delta\omega = |\omega_s - \omega_i| = \omega(\mathbf{q})$（声子频率）
- 入射光波矢 $\mathbf{k}_i$ 和散射光波矢 $\mathbf{k}_s$ 的大小和方向（由散射几何确定） → 波矢差 $\Delta\mathbf{k} = \mathbf{k}_s - \mathbf{k}_i = \pm\mathbf{q} + \mathbf{K}_h$

**得出色散关系的方法**：改变散射角 $\theta$（入射光与散射光之间的夹角），每次测量给出一个 $(\omega, \mathbf{q})$ 数据点。扫描所有散射角（即扫描所有 $\mathbf{q}$），即可得到完整的色散曲线 $\omega(\mathbf{q})$。

**两种主要实验技术**：
- **拉曼散射（Raman scattering）**：探测光学支声子。$\mathbf{k}_i \approx \mathbf{k}_s \approx 0$（可见光波长远大于晶格常数），因此只能探测 $q \approx 0$ 的光学声子 → 测量 $\omega_O(q=0)$
- **布里渊散射（Brillouin scattering）**：探测声学支声子。同样 $q \approx 0$ → 测量长波声学声子频率 → 可得声速 $v_s = \omega/q$

要测量 $q \neq 0$ 的声子，需使用**非弹性X射线散射（IXS）**或**非弹性中子散射（INS）**——中子波长与原子间距可比拟，波矢覆盖整个布里渊区，可完整测量色散关系 $\omega(\mathbf{q})$。

# 第四章 金属自由电子理论

#### 1. 如何描述自由电子气体的基态？解释费米面的物理含义。
自由电子气体模型将金属中的价电子看作是无相互作用的费米子，被限制在体积$V=L^3$的立方体内，并忽略离子实的周期性势场（自由电子近似）。系统的基本态（绝对零度$T=0K$）对应所有电子尽可能占据最低的能态，同时遵从泡利不相容原理。在周期性边界条件下，电子波矢为$\mathbf{k}=\frac{2\pi}{L}(n_x,n_y,n_z)$，能量为$\varepsilon_\mathbf{k}=\frac{\hbar^2k^2}{2m}$。每个$\mathbf{k}$态可容纳自选相反的两个电子。基态时，电子从最低能量态$(k=0)$开始填充，直到将$N$个电子排完，在$k$空间中形成一个球——费米球。
**费米面的物理含义**：波矢空间中，被电子占据的状态与为被电子占据的状态的分界面$(E=E_F)$。
**费米波矢$k_F$**：费米球的半径，由电子数半径$n=\frac{N}{V}$决定
$$
n=\frac{2}{(2\pi)^3}\cdot\frac{4}{3}\pi k_F^3\to k_F=(3\pi^2n)^\frac{1}{3}
$$
**费米能量$E_F$**：费米面上的电子能量，是基态的最高能量。
$$
E_F=\frac{\hbar^2k_F^2}{2m}
$$
**费米速度$v_F$**：费米面上电子的速度 $v_F=\frac{\hbar k_F}{m}$
**总能量$E_0$**：基态所有电子能量之和
$$
E_0=\frac{3}{5}NE_F
$$

#### 2. 为什么一般情况下只有费米面附近的电子能被激发？
$T\ne0$时，自由电子服从费米狄拉克分布
$$
f(k)=\frac{1}{e^{(E-E_F)/k_BT}+1}
$$
$T=0$的时候为阶跃函数。
一般情况下（$k_BT\ll E_F$，弱外场，低光子能量），<mark style="background:#fff88f">电子获得的激发能远小于费米能</mark>，而且泡利不相容原理禁止电子跃迁到已被占据的态。因此，只有那些初识能量位于费米面附近$~k_BT$范围内的电子，才有机会跃迁到费米面以上的空态。  
#### 3. 为什么基态下电子的动能不为零？为什么自由电子气体的密度越大，电子的平均动能就越大？
在绝对零度下，电子的平均动能与费米能级有相同的数量级，原因在于电子服从泡利不相容原理，绝对零度时，不可能发生所有电子都集中在最低能态上的情况。
$$
n=\frac{2}{(2\pi)^3}\cdot\frac{4}{3}\pi k_F^3\to k_F=(3\pi^2n)^{\frac{1}{3}}
$$
$$
E_F^0=\frac{\hbar^2k_F^2}{2m}=\frac{\hbar^2}{2m}(3\pi^2n)^\frac{2}{3}
$$
$$
E=\int_0^{E_F^0}E\rho(E)dE=\int_0^{E_F^0}CE^\frac{3}{2}=\frac{3}{5}NE_F^0
$$
当粒子数密度$n$增大时，意味着单位空间内的电子更多。为了把这么多电子塞进有限的动量空间且不违反泡利不相容原理，必然要占据更高波矢的量子态。

#### 4. 金属中自由电子气体的比热比经典理论给出的结果小很多，为什么？
按照经典理论，每个电子的平均动能为$\frac{3}{2}k_BT$，对比热的贡献为$\frac{3}{2}k_B$。
在金属中，电子是费米子，遵循泡利不相容原理。在绝对零度下，电子从最低能级开始填充，直到费米能级$E_F^0$，所有低于费米能级的能级都被占满，形成费米海洋。实际上只有费米面附近几个$k_BT$附近的电子有贡献，深层的电子被冻结了，参与热激发的电子比例
$$
\frac{可激发电子数}{总电子数}\sim\frac{k_BT}{E_F}=\frac{T}{T_F}
$$

#### 5. 晶体的比热主要来源于电子的比热和晶格振动的比热，这两部分比热各自与温度有什么关系？为什么低温时晶格振动比热主要来源于长波声学支声子的贡献，而电子比热主要来源于费米面附近电子的贡献？
**晶格振动比热**：（德拜模型）
高温极限($T\gg\Theta_D$，$\Theta_D$为德拜温度)：所有振动模式都被激发，遵循杜隆-珀蒂定律
$$
C\approx3N_Ak_B
$$
低温极限（$T\ll\Theta_D$）：只有长波声学支被激发，比热按温度的立方律下降
$$
C_l=\frac{12\pi^4}{5}N_Ak_B\left(\frac{T}{\Theta_D}\right)^3\propto T^3
$$
**电子比热**：（索末菲模型）
电子比热在整个低温区间（$T\ll T_F$)
$$
C_e=\gamma T=\frac{\pi^2}{2}N(E_F)k_B^2T\propto T
$$
1. 声学支和光学支的频谱差异：声学支频率$\omega$随波矢线性变化（$\omega=\nu_s q$），在布里渊区中心（$q\to0$）频率区域0，没有激发阈值；光学支频率在布里渊区中心处有限，存在一个能量阈值$\hbar\omega_0$
2. 低温热激发的能量尺度：低温下，声子的平均热能约为$k_BT$。只有当声子能量$\hbar\omega<k_BT$时，该模式才能被有效激发

在绝对零度下，所有能量低于$E_F^0$的量子态全部被电子填充，高于费米能级的全空。当温度升高的有限$T$时，热扰动仅仅能影响在费米面$\sim k_BT$范围内的电子。
#### 6. 为什么实际金属中的电子会受到散射，散射的主要来源有哪些？
晶体中严格的周期性势场被破坏了。理想晶体中，布洛赫电子在周期势能场中运动，波函数为扩展态，不会发生散射（电阻为0）。真实的金属中存在各种缺陷和晶格振动，这些因素破坏了平移对称性，导致电子的晶格动量$\hbar\mathbf{k}$发生改变，产生电阻。
- 杂质原子：替位式或间隙式杂质
- 晶格缺陷：空位、位错、层错、缺陷、晶界
- 表面和界面散射（尺寸效应）：当金属薄膜或纳米线的厚度小于电子的平均自由程时，电子与样品表面或晶界发生碰撞。
- 声学支声子散射：温度升高时，原子振动振幅加大，晶格势场的偏离量增加。电子在运动时被这些振动的原子散射。
- 光学支声子散射：在非简单金属（如过渡金属）或化学物中，光学支声子能量较高。只有在极高温度下，光学支声子大量激发时，这部分散射才会显著贡献。
- 电子——电子散射：一个电子受到另一个电子的库伦排斥，能量和动量发生交换。
#### 7. 有没有可能实现理想导体（即电阻为零），如果可以，你认为如何实现？
常规金属即使在极低温下，因杂质和缺陷的存在，电子仍会被散射，存在残余电阻。要实现绝对的0电阻，需要依靠宏观量子效应（超导或拓扑保护）。
- 超导态（BCS理论）：在极低温下，电子通过交换声子形成库珀对。库珀对作为玻色子凝聚到一个宏观的量子态上，其激发谱存在超导能隙。只要热涨落的能量小于该能隙，库珀对就无法被拆散，散射被严格禁止，从而实现零电阻。
- 拓扑边界态（量子霍尔效应）：在强磁场或磁性拓扑材料中，电子处于手性边缘态。这种态的电子只能单向运动，其背向散射在物理上被拓扑对称性禁止。

#### 8. 阐述电场对费米球的影响，为什么说主要是费米面处电子对电输运有贡献？
在$\mathbf{k}$空间，电场引起的漂移速度对应于波矢$\mathbf{k}$的改变
$$
\mathbf{v}_d=-\frac{e\tau\mathbf{E}}{m}\to\hbar\Delta\mathbf{k}=-e\mathbf{E}\tau
$$
即电子气中所有电子在$\mathbf{k}$空间平移了$\Delta k$。如果没有碰撞，在电场的作用下，在$\mathbf{k}$空间中费米球将沿着$-\mathbf{E}$方向不断漂移，对应于$\mathbf{v}$的持续增加。碰撞破坏了这种过程，导致费米球稳定在偏离平衡的新位置上，想应于有确定的漂移速度$\mathbf{v}_d$。
非平衡费米球中于$E=0$时费米球交叠部分，由于在$\pm\mathbf{k}$方向分布的对称性，对电流没有贡献。电流来源于相对于原费米球，$+k_F$附近态的占据有所增加，$-k_F$附近有所减少，占据有非均衡的变化。
$$
J=-e\left(n\frac{v_d}{c_F}\right)v_F
$$

#### 9. 金属中的哪些电子可以从表面逃逸出来？

电子从金属表面逃逸（热电子发射或光电发射）需要满足两个条件：

**（1）能量条件**：电子垂直于表面方向的动能必须大于功函数 $W$。

$$E_\perp = \frac{\hbar^2 k_\perp^2}{2m} > E_F + W$$

其中 $E_F$ 为费米能，$E_F + W$ 为真空能级（$E_0$）。

**（2）动量/方向条件**：只有那些速度方向朝向表面、且法向动能分量的电子才能克服表面势垒。并非所有能量高于 $E_F+W$ 的电子都能逃逸——运动方向远离表面的电子即使总动能足够大也无法逃逸。

**可以逃逸的电子特征**：
- 能量远高于费米能 $E_F$（位于费米-狄拉克分布的高能尾部）
- 在费米分布中，$E - E_F \gg k_BT$ 时分布函数 $f(E) \approx e^{-(E-E_F)/k_BT}$（玻尔兹曼尾部），因此可逃逸电子数正比于 $e^{-W/k_BT}$
- 这解释了热发射电流的里查逊-杜什曼公式 $j = AT^2 e^{-W/k_BT}$ 中的指数温度依赖性

实际上，能够逃逸的电子是费米分布高能尾部的极少数电子，它们的热运动动能异常大（$\sim E_F + W$），占比约为 $e^{-W/k_BT}$。室温下 $W \sim 2\text{–}5\,\text{eV} \gg k_BT \approx 0.025\,\text{eV}$，可逃逸电子极少。这也是为什么需要加热（热发射）或光照（光电发射）才能获得可观测的发射电流。

#### 10. 计算热发射电子流时为什么可以采用玻尔兹曼统计来近似计算？

**原因**：能够克服功函数 $W$ 从金属表面逃逸的电子，其能量 $E \ge E_F + W \gg E_F$。在费米-狄拉克分布函数

$$f(E) = \frac{1}{e^{(E-E_F)/k_BT} + 1}$$

中，当 $E - E_F \gg k_BT$（对热发射电子，$E - E_F \ge W \gg k_BT$ 严格成立）时：

$$e^{(E-E_F)/k_BT} \gg 1 \Rightarrow f(E) \approx e^{-(E-E_F)/k_BT}$$

**费米分布退化为玻尔兹曼分布**。因此可以直接用玻尔兹曼统计（麦克斯韦-玻尔兹曼速度分布）来计算热发射电子流，既准确又大幅简化计算。

**更广泛的理解**：这对应经典极限条件——当量子态上的平均占据数 $\langle n\rangle \ll 1$ 时，量子统计（费米/玻色）过渡到经典玻尔兹曼统计。热发射电子的能量远在费米能级以上，该能区的态密度虽然很大但电子占据概率极低（$\sim e^{-W/k_BT} \ll 1$），满足经典极限条件。

最终得到的里查逊-杜什曼公式 $j = \frac{4\pi mek_B^2}{h^3}T^2 e^{-W/k_BT} = AT^2 e^{-W/k_BT}$ 就是直接用玻尔兹曼统计计算的结果。

#### 11. 两种不同金属紧密接触后，为什么会产生接触电势差？接触电势差与金属的什么性质有关？
接触电势差的来源是费米能级之差，电子从费米能级高的金属流向费米能级低的金属，到达平衡时，两边费米能级一样高.（理解：多了电子、费米能级自然升高、电子少了、费米能级自然降低）这个结论符合统计物理学基本原理，对于粒子数可变系统，平衡条件是化学势（即费米能）相等.

# 第五章 能带理论

#### 1. 固体能带理论的基础是周期场中运动的单电子模型，将晶体中相互作用的电子和原子核构成的多粒子体系简化成周期场中运动的单电子模型作了哪几种近似？阐述这些近似的物理含义。
- 绝热近似：由于原子核质量比电子的质量大得多，电子的运动速度远远大于原子核的运动速度，即认为原子核的运动跟不上电子的运动。因此在考虑电子的运动时，认为<mark style="background:#fff88f">原子实不动</mark>，多粒子系统简化为相互作用的多电子系统。
- 单电子近似：<mark style="background:#fff88f">一个电子在离子实和其他电子所形成的势场中运动</mark>。又称作Hartree-Fock近似。相互作用多电子系统简化为无相互作用的电子系统
- 周期近似：<mark style="background:#fff88f">原子实和电子所形成的势场是周期性的</mark>。周期势场中无相互作用的电子系统。

#### 2. 晶体中电子体系的哈密顿量可写成$\hat{H}_e=-\sum_i\frac{\hbar^2}{2m}\nabla^2_i+\frac{1}{2}\sum_{i\ne j}\frac{e^2}{|r_i^2-r_j^2|}+\sum_{i,l}V(\mathbf{r_i}-\mathbf{R_l})$，其中 $\mathbf{r}_i$ 和 $\mathbf{R}_l$ 分别表示电子和离子实的位置。从上述哈密顿量出发，论述金属自由电子气体模型和固体能带理论的物理思想，写出这两种模型的哈密顿量。
1. 金属自由电子气体模型：
- 凝胶模型：<mark style="background:#fff88f">忽略电子和离子实间的相互作用，将离子实看作是正电荷均匀分布的背景。</mark>
- <mark style="background:#fff88f">忽略电子-电子间的相互作用，既独立电子近似</mark>。描述自由电子气体的独立参量：电子数密度n
$$
\hat{H}=\sum_{i=1}^N\left(-\frac{\hbar^2}{2m}\nabla^2_i\right)
$$
哈密顿量的本征态是平面波$\psi_\mathbf{k}(\mathbf{r})\sim e^{i\mathbf{k}\cdot\mathbf{r}}$，本征能量为$E(\mathbf{k})=\frac{\hbar^2 \mathbf{k}^2}{2m}$，引出了费米球和费米面的概念
2. 固体能带理论
$$
\hat{H}_{band}=\sum_{i=1}^N\left[-\frac{\hbar^2}{2m}\nabla_i^2+\sum_lV(\mathbf{r_i}-\mathbf{R_i})\right]
$$
自由电子气模型丢掉了一切势场，只保留了动能，适用于简单金属的宽泛宏观性质；能带理论保留并强调了势场的周期性，是理解固体多样性的基石。

#### 3. 在周期性势场中运动的粒子，其本征态和本征值有什么特点。

**本征态（布洛赫波函数）的特点**：

$$\psi_{n\mathbf{k}}(\mathbf{r}) = e^{i\mathbf{k}\cdot\mathbf{r}} u_{n\mathbf{k}}(\mathbf{r}),\quad u_{n\mathbf{k}}(\mathbf{r}+\mathbf{R}) = u_{n\mathbf{k}}(\mathbf{r})$$

1. **调幅平面波形式**：平面波 $e^{i\mathbf{k}\cdot\mathbf{r}}$ 被一个与晶格同周期的函数 $u_{n\mathbf{k}}$ 调制。这意味着电子在晶体中既不是完全自由（纯平面波），也不是完全局域（原子轨道），而是介于两者之间——在原子尺度被周期势调制，在宏观尺度保持波的传播特性。

2. **倒格子空间周期性**：$\psi_{n\mathbf{k}+\mathbf{K}_h} = \psi_{n\mathbf{k}}$。$\mathbf{k}$ 与 $\mathbf{k}+\mathbf{K}_h$ 描述同一个物理状态。因此 $\mathbf{k}$ 只需限制在第一布里渊区内。

3. **$\hbar\mathbf{k}$ 是准动量（晶体动量）**：它不是真实的动量（布洛赫波不是动量算符的本征态）。$\hbar\mathbf{k}$ 是平移算符的本征值，反映了电子在周期结构中"传播"的量子数。

4. **能带指标 $n$ 的来源**：对于每一个 $\mathbf{k}$，薛定谔方程有无穷多个分立解，用 $n$ 标记。$n$ 本质上来源于原子能级——在紧束缚图像中，每个孤立原子能级展宽为一个能带。

**本征值（能量）的特点**：

1. **能带结构**：$E = E_n(\mathbf{k})$。自由电子的连续抛物线 $E = \hbar^2 k^2/2m$ 被周期势劈裂为一系列被禁带（带隙）隔开的能带。

2. **倒格子空间周期性**：$E_n(\mathbf{k}+\mathbf{K}_h) = E_n(\mathbf{k})$。能量在第一布里渊区内是 $\mathbf{k}$ 的多值函数（不同 $n$ 对应不同能带）。

3. **能带宽度**：由原子间波函数交叠（迁移积分 $J$）决定。原子间距越小、波函数交叠越大，带宽越大。

4. **带隙形成**：在布里渊区边界，$|\mathbf{k}| = |\mathbf{k}-\mathbf{K}_h|$，简并的平面波通过周期势混合，形成驻波（成键态和反键态），两者能量差为带隙 $E_g = 2|U(\mathbf{K}_h)|$。

5. **在 $\mathbf{k}$ 空间的对称性**：$E_n(\mathbf{k})$ 具有晶体点群的全部对称性。$\mathbf{k}=0$（$\Gamma$ 点）和布里渊区边界的高对称点通常是能带的极值点。

6. **态密度分布**：能带平坦处（$\nabla_{\mathbf{k}}E \approx 0$，范霍夫奇点）态密度极大；能带陡峭处态密度小。

#### 4. 阐述布洛赫定律。布洛赫波函数中波矢 $\mathbf{k}$ 的物理含义是什么？一个布里渊区波矢 $\mathbf{k}$ 有多少个取值？一个能带能容纳多少个电子？
Bloch定理：在周期性势场中运动的电子，其波函数具有如下形式：
$$
\psi_{n\mathbf{k}}(\mathbf{r})=e^{i\mathbf{k}\cdot\mathbf{r}}u_{n\mathbf{k}}(\mathbf{r})
$$
其中$u_{n\mathbf{k}}(\mathbf{r}+\mathbf{R})=u_{n\mathbf{k}}(\mathbf{r})$是一个具有晶格周期性的函数。
周期性边界条件：$\psi(\mathbf{r}+N_i\mathbf{a}_i)=\psi(\mathbf{r})=e^{i\mathbf{k}\cdot N_i\mathbf{a_i}}\psi(\mathbf{r})$
k的取值有N个，N为原胞数，一个能带能容纳2N个电子，每个k态可以容纳2个。

#### 5. 从近自由电子近似的角度出发，理解为什么在布里渊区边界会形成带隙。
在布里渊区边界，周期势场使自由电子发生布拉格反射，简并的平面波态叠加成驻波。由于余弦驻波将电子集中到原子附近（低势能），正弦驻波将电子集中到原子间隙（高势能），两者的能量差$(2|V_n|)$便在能量谱中打开一个没有量子态的禁带。势场越强，带隙越宽。
#### 6. 平面波近似和紧束缚近似是计算晶体电子能带结构的两种方法，阐述这两种方法的物理思想以及它们各自所适用的对象。
- 平面波近似：核心假设是电子在晶体中几乎是自由的，把周期势场看作是很弱的微扰。在这种图像中，离子实对电子的吸引很弱，电子更像是在一个均匀的正电荷背景中运动，原子实的周期排列仅对电子波产生轻微的“调制”。电子的行为接近真空中的自由电子，其波函数接近平面波。
适用对象：价电子高度巡游，束缚很弱；原子实对电子的散射很弱，势能变换平缓；费米面基本接近球体，真实能带与自由电子的抛物线偏离不大。
- 紧束缚近似：核心假设是电子在原子附近受到非常强的束缚，周期势场不能视作微扰。在这种图像中，电子被牢牢束缚在单个原子核周围，其波函数与孤立原子中的轨道波函数（如1s,2p）十分接近。当原子结合成晶体时，电子在相邻原子之间发生微弱的跳跃，从而摆脱了绝对的局域，形成扩展的能带。原子能级因原子间波函数重叠而分裂形成能带。
适用对象：过渡金属（d电子）、半导体、绝缘体以及层状/分子晶体

#### 7. 晶体中电子费米面的形状和费米面处的态密度对晶体的性质有很大的影响，为什么？
在金属和半金属中，几乎所有的低能物理过车过都只由费米面附近（$\sim k_BT$能量范围内）的电子决定。
1. 费米面是$\mathbf{k}$空间中能量等于费米能$E_F$的等能面。它的形状直接决定了电子的群速度$(\mathbf{v}_n(\mathbf{k})=\frac{1}{\hbar}\nabla_\mathbf{k}E_n(\mathbf{k}))$、有效质量（$\left(\frac{1}{m^*}\right)=\frac{1}{\hbar^2}\frac{\partial^2E}{\partial k_i\partial k_j}$)和散射机制
2. 态密度定义为费米面处单位能量间隔的量子态数目。
- 决定电子比热：电子比热系数$\gamma=\frac{\pi^2}{3}k_B^2N(E_F)$。低温比热为$C=\gamma T+\beta T^3$，如果态密度很大，电子比热显著增大，在极低温下甚至超过晶格比热
- 决定泡利顺磁性：磁化率$\chi=\mu_o\mu_B^2N(E_F)$。费米面处态密度越大，自旋向上和自旋向下的能级在外磁场分裂后，能够反转自旋饼转移到高能态的电子就越多，磁化强度就越大。
- BCS理论中，超导转变温度$T_c\propto\Theta_D\exp(\frac{1}{N(E_F)V})$，态密度太低，无法形成足够的库珀对，超导难以发生

#### 10. 如何理解电子的负有效质量？
电子的负有效质量是晶体周期势在能带顶部对电子施加反向晶格力超过外力时，为保留牛顿定律数学形式而出现的等效参数。它本质上是<mark style="background:#fff88f">原子晶格强烈散射电子的宏观表现</mark>，而解决这一反直觉现象的最佳物理方案就是用带正电荷、正有效质量的空穴来替代带顶的电子。
#### 11. 根据固体能带理论阐明满带电子和不满带电子的导电行为，描述导体、半导体和绝缘体的能带结构。
$$
\mathbf{J}=-e\int_{occ}\frac{1}{4\pi^3}\mathbf{v}(\mathbf{k})d^3\mathbf{k}
$$
$$
\frac{d\mathbf{k}}{dt}=-\frac{1}{\hbar}q\mathbf{E}
$$
- **满带电子不导电**，当有外场时，每个电子波矢随时间变化，但不考虑能带间的跃迁，波矢$\mathbf{k}$和$\mathbf{k}+\mathbf{K}_h$是等价的，满带状态没有改变
- **不满带的电子导电**，无外场时电流密度也为0。在外场作用下，整个电子分布将向一方移动，破坏了原来的对称分而产生一个偏移，从而产生电流。不满带能带称为导带。
#### 12. 阐述为什么金属具有良好的导电性而绝缘体不导电。本征半导体具有与绝缘体类似的能带结构，但为什么具有一定的导电性？
- 金属导电是因为在费米能级附近，始终存在大量未被占据的量子态（空态），这些空态与占据态之间没有能量间隔。费米能级落在了能带内部，电子不需要能量就可以找到空态。
- 绝缘体不导电是因为价带是满带，导带是全空的，但是价带和导带之间的禁带宽度太大。费米能级位于禁带正中央。几乎没有电子能够被激发到导带。
- 本征半导体在绝对零度时其能带结构完全等同于绝缘体：价带全满，导带全空，费米能级位于禁带中央。但是禁带宽度较窄为0.1eV-0.3eV之间，价带电子可以通过激发到导带，形成导电电子，电子跃迁后留下的空穴也能导电。
#### 13. 简述导体、绝缘体、半导体的电子结构，在此基础上说明导体、绝缘体和半导体的导电行为。
**电子结构**
- 导体：存在部分填充的能带，或导带和价带发生能量重叠，费米能级落在允带，附近同时存在占据态和空态
- 绝缘体：<mark style="background:#fff88f">价带全满，导带全空</mark>。费米能级落在禁带中央，远离导带和价带，<mark style="background:#fff88f">禁带很宽</mark>。
- 半导体：<mark style="background:#fff88f">价带全满，导带全空</mark>。费米能级落在禁带中央，<mark style="background:#fff88f">禁带窄</mark>。
**导电行为**
- 导体：无需能量门槛，载流子多，电阻率极低，且温度升高时电阻率增大，因为晶格振动剧烈，散射加强。
- 绝缘体：能量壁垒太高，电子被"冻结“在价带
- 半导体：有限温度下的可控激发，禁带宽度窄，室温下就可以激发出一定的载流子浓度。电导率随温度的升高呈指数增长。

#### 14. 阐述描述晶体中电子在外场作用下的半经典近似的物理思想。根据半经典近似描述晶体中电子在稳恒电场中的运动，并根据固体能带理论阐明满带电子和不满带电子的导电行为。
**物理思想**：晶体中的电子受到外场作用时，一般情况下外场势远小于晶体的周期势，对晶体周期势采用能带论的量子力学处理，即每个电子具有确定的波函数、波矢和给定的能量$E(K)$，外场对晶体采用经典物理的方式来处理。把电子看出波包。
**半经典近似下电子在稳恒电场（$E$）中的运动**
$$
\frac{d(\hbar\mathbf{k})}{dt}
=\mathbf{F}\to\mathbf{k}(t)=\mathbf{k}(0)-\frac{e\mathbf{E}}{\hbar}t
$$
在没有散射的情况下，$\mathbf{k}$随时间线性增加。当$\mathbf{k}$到达布里渊区边界时，它会在该边界上发生布拉格反射，等价于从边界另一侧重新进入布里渊区。这种周期运动导致电子在实空间做布洛赫振荡，直流的外加电场可能产生交变的电流。在一个完整周期内，电子的平均漂移速度为0，完美周期晶体家直流电场并不产生真正的直流电流（绝热条件下）。但是真实晶体中都会存在杂志和缺陷，会发生散射。

#### 16. 什么是朗道轨道？朗道轨道是怎么样形成的？有什么特点？

**朗道轨道**：电子在强磁场中做回旋运动的量子化轨道。在垂直于磁场的平面内，电子的连续动能谱被量子化为分立的**朗道能级** $E_\nu = (\nu + \frac{1}{2})\hbar\omega_c$（$\omega_c = eB/m_c$ 为回旋频率），每个朗道能级在 $\mathbf{k}$ 空间中对应一个等能环——称为**朗道环**（Landau ring），其实空间的对应物即为**朗道轨道**。

**形成机制**：

1. **经典层面**：电子在垂直磁场中受洛伦兹力做匀速圆周运动（回旋运动），回旋频率 $\omega_c = eB/m$，回旋半径 $r_c = mv_\perp/eB$。

2. **半经典层面**：在 $\mathbf{k}$ 空间中，电子沿等能面与垂直 $\mathbf{B}$ 的平面的交线运动。轨道量子化条件（玻尔-索末菲）：

$$\oint \mathbf{p}\cdot\mathrm{d}\mathbf{r} = \left(\nu + \frac{1}{2}\right)h \Rightarrow S(\varepsilon_\nu) = \frac{2\pi eB}{\hbar}\left(\nu + \frac{1}{2}\right)$$

其中 $S$ 为轨道在 $\mathbf{k}$ 空间的面积。这意味着只有面积满足量子化条件的那些 $\mathbf{k}$ 空间轨道才是允许的——原本在垂直 $\mathbf{B}$ 平面内准连续的 $\mathbf{k}$ 点被"压缩"成一系列**朗道管道**（Landau tubes），管道截面即为朗道环。

3. **量子力学层面**：引入磁矢势，薛定谔方程简化为谐振子方程。自由电子在垂直磁场平面内的动能 $\hbar^2(k_x^2+k_y^2)/2m$ 被量子化为 $\hbar\omega_c$ 的等间距能级。

**特点**：

1. **等间距能级**：$E_\nu = (\nu + \frac{1}{2})\hbar\omega_c$，能级间距 $\hbar\omega_c = \hbar eB/m_c \propto B$。磁场越强，能级间距越大。

2. **极高的简并度**：每个朗道能级的简并度 $n_B = eB/h$（单位面积）。物理原因：磁场使原本在 $\mathbf{k}$ 空间中分布在 $2\pi eB/\hbar$ 面积内的连续态简并为一个朗道能级。$B = 1\,\text{T}$ 时 $n_B \approx 2.4 \times 10^{10}\,\text{cm}^{-2}$。

3. **自旋分裂**：计入电子自旋与磁场的塞曼耦合 $g\mu_B B\sigma_z$，每个朗道能级进一步分裂为两个自旋子能级，间距为 $g\mu_B B$（通常 $\ll \hbar\omega_c$）。

4. **$\mathbf{k}$ 空间的体现**：许可态聚集在朗道环上，朗道环截面面积随 $\nu$ 线性增大。当磁场变化时，朗道环扫过费米面 → 产生**德哈斯-范阿尔芬效应**和**Shubnikov-de Haas效应**的量子振荡。

5. **与量子霍尔效应的关系**：朗道能级的填充因子 $\nu_{\text{fill}} = n/n_B$ 决定霍尔电导的平台位置。整数（分数量子）霍尔效应分别对应朗道能级的整数（分数）填充。

6. **轨道形状**：朗道轨道在实空间是闭合的圆形（对称规范）或沿一个方向延伸的条带状（朗道规范），但两种描述物理等价。轨道中心的引导中心（guiding center）对应 $\mathbf{k}$ 空间中轨道的位置。
