# 固体物理期末考试复习计划

**目标**：期末考试高分。考试不超纲，内容来源于PPT、作业题、教材、思考题。

**已有复习资源**：
- `固体物理学复习笔记.md` — 五章完整知识点
- `固体物理思考题.md` — 全部思考题含解答（54道）
- `固体物理作业题汇总.md` — 全部作业题（26道，含第五章第一题解答）
- 各章PPT + 知识点文档

---

## 总体策略

**高分关键**：

1. **思考题是第一优先级** —— 老师出题最直接的来源，54道思考题覆盖全部考点。考前应做到：看到题目就能说出完整答案
2. **作业题是第二优先级** —— 计算题和证明题的直接模板，考试中的计算题大概率是作业题的变体
3. **核心推导必须能手写** —— 克拉珀龙方程、色散关系、费米能级、紧束缚能带、有效质量等
4. **物理图像 > 死记公式** —— 理解"为什么"比记住"是什么"更重要，问答题占分通常不低

**每日安排原则**：上午新内容（概念+推导），下午做题巩固（思考题+作业题），晚上快速回顾+标记薄弱点。

## Day 3：第三章 晶格振动动力学（上）

**时间分配**：约5–6小时（本章最重，分两天）

**上午——一维链模型（2.5h）**
- 阅读复习笔记第三章 3.1–3.3
- 重点：
  - **一维单原子链色散关系推导**：$\omega(q) = 2\sqrt{c/m}\,|\sin(qa/2)|$，从运动方程到色散关系的完整推导
  - **一维双原子链色散关系推导**：声学支和光学支的区别
  - $q \to 0$ 和 $q = \pi/2a$ 极限行为
  - 周期性边界条件 → $q$ 取分立值 → 模式计数
  - 声子的概念：玻色子、$\mu=0$

**下午——思考题（2–2.5h）**
- 重点攻克：Q2（周期性边界条件）、Q3（双原子链格波性质）、Q5（模式计数）、Q7（声子）
- Q3必须能手写完整推导

**晚上——回顾+自测（30min）**

## Day 4：第三章 晶格振动动力学（下）

**时间分配**：约5小时

**上午——比热与非简谐效应（2h）**
- 阅读复习笔记第三章 3.4–3.6
- 重点：
  - **爱因斯坦模型**：假设、热容公式、$\Theta_E$、低温 $C_V \propto e^{-\Theta_E/T}$
  - **德拜模型**：连续介质近似、态密度 $\propto \omega^2$、$\Theta_D$、低温 $C_V \propto T^3$
  - **为什么德拜优于爱因斯坦？**——长波声学模无能隙，这是必考题
  - 简谐近似的局限性 + 非简谐效应（热膨胀的微观来源、热阻的Umklapp过程）

**下午——思考题+作业题（2.5h）**
- 思考题 Q9（爱因斯坦vs德拜）、Q10（简谐vs非简谐）、Q11（非弹性散射测色散关系）
- 作业题第2题（$v_p$ 和 $v_g$ 曲线）、第4题（正方格子色散关系）——重要
- Q6（为什么离子晶体长光学波最重要）——理解极化激元概念

**晚上——第三章总结（30min）**
- 梳理第三章所有公式：$\omega(q)$、$\langle n \rangle$、$C_V$（爱因斯坦和德拜）

## Day 5：第四章 金属自由电子理论（上）

**时间分配**：约5小时

**上午——自由电子气体模型与基态（2.5h）**
- 阅读复习笔记第四章 4.1–4.4
- 重点：
  - **自由电子气体模型的基本假设**
  - **费米球的推导**：$k_F = (3\pi^2 n)^{1/3}$（从 $\mathbf{k}$ 空间态密度出发的完整推导）

**下午——思考题+作业题（2.5h）**

## Day 7：第五章 能带理论（上）

**时间分配**：约5–6小时（本章最抽象，需要最多时间理解）

**上午——单电子近似与布洛赫定理（2.5h）**
- 阅读复习笔记第五章 5.1–5.7
- 重点：
  - **三个近似的物理含义**：绝热近似 → 平均场近似 → 周期场近似（这是必考题）
  - **布洛赫定理**：$\psi_{\mathbf{k}}(\mathbf{r}) = e^{i\mathbf{k}\cdot\mathbf{r}}u_{\mathbf{k}}(\mathbf{r})$，$\mathbf{k}$ 的物理含义
  - 能带形成的物理图像：原子能级 → 能带
  - **近自由电子近似**：布里渊区边界形成带隙的原因（布拉格反射 → 驻波 → 能隙）
  - **紧束缚近似**：$E(\mathbf{k}) = \varepsilon_0 - J_0 - \sum J(\mathbf{R})e^{i\mathbf{k}\cdot\mathbf{R}}$，各项物理含义
  - 平面波方法 vs 紧束缚方法的适用对象对比

**下午——思考题（2.5h）**
- 重点攻克：Q1（三种近似）、Q3（本征态和本征值特点）、Q4（Bloch定理）、Q5（带隙形成）、Q6（两种方法对比）
- 这些是第五章最核心的概念题

**晚上——理解巩固（30min）**
- 梳理：近自由电子 → 能带和带隙的来源
- 梳理：紧束缚 → 原子能级如何展宽为能带

## 附：考场策略

1. **先做会做的**，计算题如果卡住先跳过
2. **概念题写满**——物理图像、公式、适用条件，多写不扣分
3. **推导题写清步骤**——从什么方程出发、用了什么近似、关键中间结果
4. **单位检查**——eV、Å、$k_B$ 的值不要搞混
