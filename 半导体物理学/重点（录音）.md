# 半导体物理学 期末复习要点（最后一课总结）

## 一、基本概念与能带理论

- **电子与空穴的有效质量**
  - 定义与物理意义（反映晶格周期场对载流子运动的影响）
  - 测量方法：**回旋共振实验**
- **直接带隙与间接带隙半导体**
  - 区别：能带极值是否在同一k点
  - 典型材料
    - 直接带隙：GaAs、InP
    - 间接带隙：Si、Ge
  - 对光跃迁的影响
- **价带与导带、空穴的运动**
  - 空穴 = 价带电子的缺失
  - 在外电场下的漂移行为等效于正电荷

## 二、杂质与掺杂

- **浅能级杂质 vs 深能级杂质**
  - 浅能级：提供载流子（如P、B在Si中）
  - 深能级：起复合中心/陷阱（如Au、Fe）
- **两性杂质**
  - 例：Si在GaAs中可取代Ga（施主）或As（受主）
  - 掺杂类型与生长条件（气氛）的关系
- **补偿效应**
  - 同时存在施主和受主时净载流子浓度的计算
- **掺杂浓度与费米能级位置**
  - n型 / p型下 EF 随温度的变化
  - 三个阶段：低温电离区 → 饱和区 → 本征区

## 三、载流子统计与电导

- **本征半导体**
  - 载流子浓度 n = p = nᵢ
  - nᵢ 随温度、禁带宽度变化（E_g 越大，nᵢ 越小，耐温性越好）
- **非本征（掺杂）半导体**
  - 饱和区载流子浓度 ≈ 净杂质浓度
  - 电导率 σ = n q μₙ + p q μₚ
- **电阻率–温度曲线**
  - 掺杂半导体三段特征：低温下降 → 饱和区平稳 → 本征区上升
  - 本质半导体单调下降
- **散射机制**
  - 晶格散射（声学波、光学波）
  - 电离杂质散射
  - 强场下可能出现负微分电导（如GaAs的RWH效应）

## 四、非平衡载流子

- **寿命 τ**
  - 少子寿命
  - 复合过程：直接复合、间接复合、俄歇复合
  - **SRH间接复合**为重点
- **复合中心能级**
  - 最有效的复合中心位于禁带中央附近
  - 深能级杂质影响寿命
- **陷阱效应**
  - 陷阱能级（靠近导带或价带）俘获一种载流子后长时间释放
  - 区别于复合中心
- **扩散与漂移**
  - 扩散系数 D 与迁移率 μ 的关系：**爱因斯坦关系** D/μ = kT/q
  - 扩散长度 L = √(Dτ)
- **连续性方程**
  - 稳态条件下少子扩散方程及其解（双极输运）

## 五、PN结

- **耗尽层宽度**、**内建电势**
  - 公式与影响因素（掺杂浓度、温度、E_g）
- **电容–电压（C–V）特性**
  - 势垒电容与偏压的关系：1/C² ~ V 线性 → 求掺杂分布
- **电流–电压（I–V）特性**
  - 理想二极管方程（扩散模型）：J = Jₛ (e^(qV/kT) - 1)
  - 正向复合电流（n=2），反向饱和电流
- **击穿机制**
  - 雪崩击穿（高反向偏压、耗尽层宽，正温度系数）
  - 隧道击穿（齐纳击穿，窄耗尽层、重掺杂，负温度系数）
  - 热击穿（不可逆）
- **肖特基结**
  - 金属–半导体接触，整流特性
  - 势垒降低（镜像力、隧道效应）

## 六、金属-半导体接触与异质结

- **欧姆接触** vs **肖特基接触**
  - 形成条件，C–V特性差异
- **异质结**
  - 能带对齐类型（Type I、II）
  - 二维电子气（如AlGaAs/GaAs）

## 七、光电性质

- **光吸收**
  - 本征吸收（光子能量 ≥ E_g）
  - 激子吸收
  - 自由载流子吸收
- **光生载流子**
  - 光伏效应
  - 光导效应

## 八、常考计算题方向

1. 根据掺杂浓度、温度计算 nᵢ、E_F、n、p
2. 利用爱因斯坦关系和扩散长度求 τ 或 D
3. PN结 C–V 曲线提取内建电势和掺杂浓度
4. 肖特基势垒高度、耗尽层宽度的计算
5. 非平衡载流子稳态分布（如光照下表面注入的少子分布）

## 九、易混淆辨析

- **直接复合 vs 间接复合**：辐射复合效率、寿命温度依赖
- **浅能级 vs 深能级**：电离能大小、对电中性影响
- **简并 vs 非简并**：E_F 进入导带/价带，杂质未完全电离，载流子统计用 Fermi–Dirac 积分
