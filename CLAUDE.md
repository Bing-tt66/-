# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 仓库性质

这是一个 **Obsidian 笔记库**，存放物理学本科课程的学习笔记、讲义 PDF 和复习资料。不是软件工程仓库。

## 目录约定

- 课程文件夹直接用课程名（如 `半导体物理学/`、`量子力学/`），平铺在根目录
- `00.索引/` — 全局 MOC（Map of Content），跨课程导航入口
- `20.概念卡片/` — 跨课程复用的物理概念卡片（如有效质量、费米-狄拉克统计），用 `[[wiki-link]]` 被多门课引用
- `90.模板/` — 新建笔记时参考的模板
- PDF/TeX/辅助文件放在对应课程文件夹内

## 笔记格式

- 正文使用中文
- 数学公式使用 LaTeX：块公式用 `$$...$$`，行内公式用 `$...$`
- 重点高亮使用 `<mark style="background:#fff88f">...</mark>`（Obsidian 风格）
- 元数据使用 YAML frontmatter（`---` 包裹，仅放在文件最开头）
- 内部链接使用 `[[笔记名]]` 或 `[[笔记名#标题]]`

## 当前工作重点

- **近期**：固体物理学、热力学与统计物理期末复习，主要需求是辅助梳理概念、推导和答疑
- **中期**：补全课程笔记，搭建从四大力学出发的完整知识体系
- **远期**：根据研究方向（待定）深入专题笔记
- 库中笔记仍有大量空白和待补内容，编辑前先确认当前文件哪些部分已完成、哪些仍在整理中

## 笔记编辑约束

- 笔记中公式可能不完整（如等号右侧留空），是因为整理未完成。**不要擅自补全**——先问
- **不要擅自修改 PDF 和 .tex 文件**
- 添加或修改笔记内容前，先阅读已有上下文，基于理论逻辑推导，不猜测
- 修改前说明方法；需求有歧义时先澄清

## Git 约定

- 仅跟踪 Markdown 笔记和 Obsidian 配置（插件 manifest、data.json）
- PDF、临时文件（`.kdtmp`、`.aux`、`.log` 等）由 `.gitignore` 排除
- 提交信息使用中文，简洁描述变更内容

## 用户背景

- 物理学本科，从事数值计算和理论模型构建推导
- 沟通使用专业语言，不需要科普式解释
- 笔记中出现的物理内容直接以专业水准讨论
