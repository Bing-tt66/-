# 第一章 热力学的基本规律

## 1.1 热力学系统的平衡状态及其描述

**热力学系统**：由大量微观粒子组成的宏观系统。

按与外界相互作用分类：
- **孤立系**：与外界既无物质交换也无能量交换
- **闭系**：无物质交换，有能量交换
- **开系**：既有物质交换又有能量交换

**热力学平衡（热动平衡）**：系统的各种宏观性质在长时间内不发生任何变化。包括力学平衡（压强均匀）、热平衡（温度均匀）、相平衡（各相质量不变）、化学平衡（各组分质量不变）。

**弛豫时间**：由初始状态达到平衡状态所经历的时间。

**状态参量**：几何参量（$V$）、力学参量（$p$）、电磁参量（$\mathcal{E}$、$\mathcal{H}$）、化学参量（$n$）。

**广延量**：与质量或物质的量成正比的量（$V$、$U$、$S$）。**强度量**：与质量或物质的量无关（$p$、$T$、$\mu$）。广延量除以质量、物质的量或体积即成为强度量。

## 1.2 热平衡定律和温度

**绝热壁**：两物体通过器壁接触，各自状态可独立改变。**透热壁**：非绝热壁。

**热平衡定律（第零定律）**：若 A 和 B 各自与同一状态的 C 达到热平衡，则 A 与 B 也将处于热平衡。

数学表述：$f_{AC}(p_A,V_A;p_C,V_C)=0 \Rightarrow p_C=F_{AC}(p_A,V_A;V_C)$，同理 $p_C=F_{BC}(p_B,V_B;V_C)$。联立消去 $V_C$：

$$g_A(p_A,V_A)=g_B(p_B,V_B)$$

此即温度相等。

**温标**：
- 气体温标：$T_V = \frac{p}{p_t} \times 273.16\,\text{K}$
- 理想气体温标：$T = 273.16\,\text{K} \times \lim_{p_t\to 0}\frac{p}{p_t}$
- 热力学温标（由第二定律导出，与测温物质无关）

## 1.3 物态方程

简单系统的物态方程：$f(p,V,T)=0$。

三个响应函数：

$$\alpha = \frac{1}{V}\left(\frac{\partial V}{\partial T}\right)_p,\quad \beta = \frac{1}{p}\left(\frac{\partial p}{\partial T}\right)_V,\quad \kappa_T = -\frac{1}{V}\left(\frac{\partial V}{\partial p}\right)_T$$

循环偏导数关系：

$$\left(\frac{\partial V}{\partial p}\right)_T \left(\frac{\partial p}{\partial T}\right)_V \left(\frac{\partial T}{\partial V}\right)_p = -1$$

由此 $\alpha = \kappa_T \beta p$。

几种典型物态方程：

| 系统 | 物态方程 |
|------|----------|
| 理想气体 | $pV = nRT$ |
| 范德瓦尔斯气体 | $(p + \frac{an^2}{V^2})(V-nb) = nRT$ |
| 昂内斯位力展开 | $p = \frac{nRT}{V}[1 + \frac{n}{V}B(T) + (\frac{n}{V})^2C(T) + \cdots]$ |
| 简单固体/液体 | $V(T,p) = V_0[1 + \alpha(T-T_0) - \kappa_T p]$ |
| 顺磁性固体 | $\mathcal{M} = \frac{C}{T}\mathcal{H}$（居里定律） |

## 1.4 功

准静态过程中外界对系统做功：

$$\mathrm{đ}W = \sum_i Y_i\,\mathrm{d}y_i$$

$y_i$ 为外参量，$Y_i$ 为广义力。常见形式：

| 系统 | $\mathrm{đ}W$ |
|------|---------------|
| 气体 | $-p\mathrm{d}V$ |
| 液体表面膜 | $\sigma\mathrm{d}A$ |
| 电介质 | $\mathbf{E}\cdot\mathrm{d}\mathbf{P}$ |
| 磁介质 | $\mu_0\mathbf{H}\cdot\mathrm{d}\mathbf{M}$ |

$\mathrm{đ}W$ 不是全微分，用 $\mathrm{đ}$ 区别于 $\mathrm{d}$。

## 1.5 热力学第一定律

$$U_B - U_A = W + Q,\quad \mathrm{d}U = \mathrm{đ}Q + \mathrm{đ}W$$

自然界一切物质都具有能量，能量在传递与转化中数量不变。第一类永动机不可能制成。

## 1.6 热容和焓

热容：$C = \lim_{\Delta T \to 0} \frac{\Delta Q}{\Delta T}$。摩尔热容 $C_m$，$C = nC_m$。

**等容热容**（$V$ 不变）：$C_V = \left(\frac{\partial U}{\partial T}\right)_V$

**等压热容**（$p$ 不变）：$C_p = \left(\frac{\partial H}{\partial T}\right)_p$

**焓**：$H = U + pV$。在等压过程中 $\Delta Q_p = \Delta H$。

## 1.7 理想气体的内能

焦耳定律：$\left(\frac{\partial U}{\partial V}\right)_T = 0$，理想气体内能只是温度的函数。

$$U = \int C_V\,\mathrm{d}T + U_0,\quad H = U + nRT$$

$$C_p - C_V = nR,\quad \gamma = C_p/C_V$$

$$C_V = \frac{nR}{\gamma - 1},\quad C_p = \gamma\frac{nR}{\gamma - 1}$$

## 1.8 理想气体的绝热过程

绝热过程 $\mathrm{đ}Q = 0$，由 $C_V\mathrm{d}T + p\mathrm{d}V = 0$ 和 $pV = nRT$：

$$\frac{\mathrm{d}p}{p} + \gamma\frac{\mathrm{d}V}{V} = 0$$

$C_V$ 为常数时：

$$pV^\gamma = \text{const},\quad TV^{\gamma-1} = \text{const},\quad \frac{p^{\gamma-1}}{T^\gamma} = \text{const}$$

声速：$a = \sqrt{\frac{\mathrm{d}p}{\mathrm{d}\rho}} = \sqrt{\gamma\frac{p}{\rho}} = \sqrt{\frac{\gamma RT}{M}}$

## 1.9 理想气体的卡诺循环

1. 等温膨胀（$T_1$）：$Q_1 = RT_1\ln\frac{V_2}{V_1}$（吸热）
2. 绝热膨胀（$T_1 \to T_2$）：$Q = 0$
3. 等温压缩（$T_2$）：$Q_2 = RT_2\ln\frac{V_3}{V_4}$（放热）
4. 绝热压缩（$T_2 \to T_1$）：$Q = 0$

由绝热过程得 $\frac{V_2}{V_1} = \frac{V_3}{V_4}$，故：

$$W = Q_1 - Q_2 = R(T_1 - T_2)\ln\frac{V_2}{V_1}$$

卡诺热机效率：$\eta = \frac{W}{Q_1} = 1 - \frac{T_2}{T_1}$

逆卡诺循环（制冷机）：$\eta' = \frac{Q_2}{W} = \frac{T_2}{T_1 - T_2}$

## 1.10 热力学第二定律

**克劳修斯表述**：不可能把热量从低温物体自发地传到高温物体而不引起其他变化。

**开尔文表述**：不可能从单一热源吸热使之完全变成有用的功而不引起其他变化。（第二类永动机不可能制成。）

两种表述等价。

## 1.11 卡诺定理

1. 所有工作于两个确定温度之间的热机中，可逆热机的效率最高
2. 所有工作于两个确定温度之间的可逆热机，其效率相等

$$\eta \leq 1 - \frac{T_2}{T_1}$$

## 1.12 热力学温标

由卡诺定理，可逆热机效率只依赖于两个热源温度。定义热力学温标使 $\frac{Q_2}{Q_1} = \frac{T_2}{T_1}$。规定水的三相点 $T_{tr} = 273.16\,\text{K}$。

## 1.13 克劳修斯等式和不等式

$$\sum_i \frac{Q_i}{T_i} \leq 0,\quad \oint \frac{\mathrm{đ}Q}{T} \leq 0$$

等号对应可逆循环，不等号对应不可逆循环。

## 1.14 熵和热力学基本方程

对可逆过程，$\int_A^B \frac{\mathrm{đ}Q}{T}$ 与路径无关，定义状态函数熵：

$$S_B - S_A = \int_A^B \frac{\mathrm{đ}Q}{T}\quad\text{（可逆过程）}$$

微分形式：$\mathrm{d}S = \frac{\mathrm{đ}Q}{T}$。

**热力学基本方程**：

$$\mathrm{d}U = T\mathrm{d}S - p\mathrm{d}V,\quad \mathrm{d}S = \frac{\mathrm{d}U + p\mathrm{d}V}{T}$$

熵是广延量：$S = S_1 + S_2 + \cdots$

## 1.15 理想气体的熵

以 $T,V$ 为自变量（$C_{V,m}$ 视为常量）：

$$S = nC_{V,m}\ln T + nR\ln V + S_0$$

以 $T,p$ 为自变量：

$$S = nC_{p,m}\ln T - nR\ln p + S_0$$

## 1.16 热力学第二定律的数学表述

对任意过程：$S_B - S_A \geq \int_A^B \frac{\mathrm{đ}Q}{T}$，微分形式 $\mathrm{d}S \geq \frac{\mathrm{đ}Q}{T}$。

**熵增加原理**：孤立系中 $\Delta S \geq 0$。不可逆绝热过程熵增加，可逆绝热过程熵不变。孤立系中不可逆过程总是朝着熵增加的方向进行。

熵是系统中微观粒子无规则运动混乱程度的量度。

## 1.17 自由能和吉布斯函数

**亥姆霍兹自由能**：$F = U - TS$。等温条件下，$-W \leq F_A - F_B$，等温等容条件下 $F$ 永不增加，平衡态 $F$ 取极小。

**吉布斯函数**：$G = F + pV = U - TS + pV$。等温等压条件下 $G$ 永不增加，平衡态 $G$ 取极小。

# 第二章 均匀物质的热力学性质

## 2.1 内能、焓、自由能和吉布斯函数的全微分

$$\begin{aligned}
\mathrm{d}U &= T\mathrm{d}S - p\mathrm{d}V \\
\mathrm{d}H &= T\mathrm{d}S + V\mathrm{d}p \\
\mathrm{d}F &= -S\mathrm{d}T - p\mathrm{d}V \\
\mathrm{d}G &= -S\mathrm{d}T + V\mathrm{d}p
\end{aligned}$$

由全微分条件得偏导数关系：

$$\begin{aligned}
\left(\frac{\partial U}{\partial S}\right)_V = T,\quad &\left(\frac{\partial U}{\partial V}\right)_S = -p \\
\left(\frac{\partial H}{\partial S}\right)_p = T,\quad &\left(\frac{\partial H}{\partial p}\right)_S = V \\
\left(\frac{\partial F}{\partial T}\right)_V = -S,\quad &\left(\frac{\partial F}{\partial V}\right)_T = -p \\
\left(\frac{\partial G}{\partial T}\right)_p = -S,\quad &\left(\frac{\partial G}{\partial p}\right)_T = V
\end{aligned}$$

## 2.2 麦克斯韦关系

由二阶混合偏导相等：

$$\boxed{\left(\frac{\partial T}{\partial V}\right)_S = -\left(\frac{\partial p}{\partial S}\right)_V}$$

$$\boxed{\left(\frac{\partial T}{\partial p}\right)_S = \left(\frac{\partial V}{\partial S}\right)_p}$$

$$\boxed{\left(\frac{\partial S}{\partial V}\right)_T = \left(\frac{\partial p}{\partial T}\right)_V}$$

$$\boxed{\left(\frac{\partial S}{\partial p}\right)_T = -\left(\frac{\partial V}{\partial T}\right)_p}$$

**记忆技巧**：由 $\mathrm{d}U$ 得对角线关系；由 $\mathrm{d}F$、$\mathrm{d}G$ 得含 $S$ 的关系。含 $S$ 和 $p$ 的关系中有一个负号。

**应用**——选 $T,V$ 为自变量，内能全微分：

$$\mathrm{d}U = C_V\mathrm{d}T + \left[T\left(\frac{\partial p}{\partial T}\right)_V - p\right]\mathrm{d}V$$

由此得：

$$C_V = T\left(\frac{\partial S}{\partial T}\right)_V,\quad \left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial p}{\partial T}\right)_V - p\quad\text{（能态方程）}$$

选 $T,p$ 为自变量：

$$\left(\frac{\partial H}{\partial p}\right)_T = V - T\left(\frac{\partial V}{\partial T}\right)_p\quad\text{（焓态方程）}$$

$$C_p - C_V = T\left(\frac{\partial p}{\partial T}\right)_V\left(\frac{\partial V}{\partial T}\right)_p = \frac{VT\alpha^2}{\kappa_T}$$

## 2.3 气体的节流过程和绝热膨胀过程

**节流过程**（焦耳-汤姆孙效应）：气体通过多孔塞从高压到低压的绝热不可逆过程，$H_1 = H_2$。

焦耳-汤姆孙系数：

$$\mu_{JT} = \left(\frac{\partial T}{\partial p}\right)_H = \frac{1}{C_p}\left[T\left(\frac{\partial V}{\partial T}\right)_p - V\right]$$

- $\mu_{JT} > 0$：致冷效应；$\mu_{JT} < 0$：致热效应
- 对理想气体，$\mu_{JT} = 0$

**绝热膨胀**：可逆绝热过程系统对外做功，温度必然降低。

## 2.4 基本热力学函数的确定

已知物态方程和 $C_V(T,V)$：

$$U = \int \left[C_V\mathrm{d}T + \left(T\frac{\partial p}{\partial T} - p\right)\mathrm{d}V\right] + U_0$$

$$S = \int \left[\frac{C_V}{T}\mathrm{d}T + \frac{\partial p}{\partial T}\mathrm{d}V\right] + S_0$$

已知物态方程和 $C_p(T,p)$：

$$H = \int \left[C_p\mathrm{d}T + \left(V - T\frac{\partial V}{\partial T}\right)\mathrm{d}p\right] + H_0$$

$$S = \int \left[\frac{C_p}{T}\mathrm{d}T - \frac{\partial V}{\partial T}\mathrm{d}p\right] + S_0$$

## 2.5 特性函数

在适当独立变量下，知道一个热力学函数即可通过求偏导得到全部热力学函数：

- $U(S,V)$、$F(T,V)$、$G(T,p)$、$H(S,p)$ 各为相应自变量的特性函数

**吉布斯-亥姆霍兹方程**：

$$\left[\frac{\partial}{\partial T}\left(\frac{F}{T}\right)\right]_V = -\frac{U}{T^2},\quad \left[\frac{\partial}{\partial T}\left(\frac{G}{T}\right)\right]_p = -\frac{H}{T^2}$$

## 2.6 热辐射的热力学理论

辐射压强 $p = \frac{1}{3}u$（$u$ 为辐射能量密度）。

由热力学关系导出斯特藩-玻尔兹曼定律：$u = aT^4$，辐射通量密度 $J = \sigma T^4$。

## 2.7 磁介质的热力学

外界对磁介质做功：$\mathrm{đ}W = \mu_0 H\mathrm{d}M$。

基本方程：$\mathrm{d}U = T\mathrm{d}S + \mu_0 H\mathrm{d}M$

相应的吉布斯函数 $G = U - TS - \mu_0 HM$：$\mathrm{d}G = -S\mathrm{d}T - \mu_0 M\mathrm{d}H$

麦克斯韦关系：$\left(\frac{\partial S}{\partial H}\right)_T = \mu_0\left(\frac{\partial M}{\partial T}\right)_H$

**磁致冷却**：$\left(\frac{\partial T}{\partial H}\right)_S = -\frac{\mu_0 T}{C_H}\left(\frac{\partial M}{\partial T}\right)_H$，对顺磁体减小磁场可致冷。

# 第三章 单元系的相变

## 3.1 热动平衡判据

**熵判据**（孤立系）：$\Delta S = \delta S + \frac{1}{2}\delta^2 S$，平衡条件 $\delta S = 0$，稳定性条件 $\delta^2 S < 0$（熵取极大）。

**自由能判据**（等温等容）：$F$ 取极小。**吉布斯判据**（等温等压）：$G$ 取极小。

平衡的稳定性条件：

$$C_V > 0,\quad \left(\frac{\partial p}{\partial V}\right)_T < 0$$

**亚稳平衡**：对无穷小变动稳定，对有限变动不稳定（如过冷液体、过热液体）。

## 3.2 开系的热力学基本方程

$$\mathrm{d}G = -S\mathrm{d}T + V\mathrm{d}p + \mu\mathrm{d}n$$

化学势：$\mu = \left(\frac{\partial G}{\partial n}\right)_{T,p}$。对单元系 $\mu = G_m$（摩尔吉布斯函数）。

各热力学势全微分：

$$\mathrm{d}U = T\mathrm{d}S - p\mathrm{d}V + \mu\mathrm{d}n,\quad \mathrm{d}H = T\mathrm{d}S + V\mathrm{d}p + \mu\mathrm{d}n$$

$$\mathrm{d}F = -S\mathrm{d}T - p\mathrm{d}V + \mu\mathrm{d}n$$

**巨热力势**：$J = F - \mu n = -pV$，$\mathrm{d}J = -S\mathrm{d}T - p\mathrm{d}V - n\mathrm{d}\mu$

## 3.3 单元系的复相平衡

两相平衡条件：$T^\alpha = T^\beta$，$p^\alpha = p^\beta$，$\mu^\alpha(T,p) = \mu^\beta(T,p)$。

## 3.4 单元复相平衡的性质

由 $\mu^\alpha(T,p) = \mu^\beta(T,p)$ 给出相平衡曲线 $p = p(T)$。

**克拉珀龙方程**：

$$\frac{\mathrm{d}p}{\mathrm{d}T} = \frac{S_m^\beta - S_m^\alpha}{V_m^\beta - V_m^\alpha} = \frac{L}{T(V_m^\beta - V_m^\alpha)}$$

其中 $L = T\Delta S_m$ 为相变潜热。

蒸气压方程（气-液平衡，气相为理想气体）：

$$\frac{\mathrm{d}\ln p}{\mathrm{d}T} = \frac{L}{RT^2},\quad \ln p = -\frac{L}{RT} + \text{const}\;(L\text{ 视为常数})$$

## 3.5 临界点和气液两相的转变

$T > T_c$ 时无法通过等温压缩使气体液化。范德瓦尔斯气体临界参数：

$$T_c = \frac{8a}{27Rb},\quad p_c = \frac{a}{27b^2},\quad V_{m,c} = 3b$$

**对应态定律**：引入对比变量 $\tilde{T}=T/T_c,\ \tilde{p}=p/p_c,\ \tilde{V}=V/V_c$：

$$\left(\tilde{p} + \frac{3}{\tilde{V}^2}\right)\left(\tilde{V} - \frac{1}{3}\right) = \frac{8}{3}\tilde{T}$$

## 3.6 液滴的形成

半径为 $r$ 的液滴附加压强 $\Delta p = 2\sigma/r$。小液滴的饱和蒸气压大于平液面的饱和蒸气压（开尔文公式），这是过饱和蒸气的成因。

## 3.7 相变的分类

**一级相变**：化学势连续，一阶偏导数不连续（有潜热、有体积突变）。

**二级相变（连续相变）**：化学势及其一阶偏导数均连续，二阶偏导数不连续（无潜热、无体积突变，$C_p$、$\alpha$、$\kappa_T$ 有突变）。

爱伦费斯特方程：

$$\frac{\mathrm{d}p}{\mathrm{d}T} = \frac{C_{p,m}^\beta - C_{p,m}^\alpha}{TV_m(\alpha^\beta - \alpha^\alpha)} = \frac{\alpha^\beta - \alpha^\alpha}{\kappa_T^\beta - \kappa_T^\alpha}$$

## 3.9 朗道连续相变理论

引入序参量 $\eta$。将自由能展开为 $\eta$ 的幂级数：

$$F(T,\eta) = F_0(T) + \frac{1}{2}a(T)\eta^2 + \frac{1}{4}b\eta^4 + \cdots$$

其中 $a(T) = a_0(T - T_c)$，$b > 0$。平衡条件 $\partial F/\partial\eta = 0$：

- $T > T_c$：$a > 0$，$\eta = 0$（对称相稳定）
- $T < T_c$：$a < 0$，$\eta = \pm\sqrt{-a/b}$（有序相稳定）

朗道理论关键结论：$C_p$ 在 $T_c$ 处有跃变；$\eta \propto (T_c - T)^{1/2}$。

# 第四章 多元系的复相平衡和化学平衡

## 4.1 多元系的热力学基本方程

$$\mathrm{d}U = T\mathrm{d}S - p\mathrm{d}V + \sum_{i=1}^{k}\mu_i\mathrm{d}n_i$$

$$\mathrm{d}G = -S\mathrm{d}T + V\mathrm{d}p + \sum_{i=1}^{k}\mu_i\mathrm{d}n_i$$

其中 $\mu_i = \left(\frac{\partial G}{\partial n_i}\right)_{T,p,n_j(j\neq i)}$ 为第 $i$ 组元的化学势。

## 4.2 吉布斯-杜安关系

$$S\mathrm{d}T - V\mathrm{d}p + \sum_i n_i\mathrm{d}\mu_i = 0$$

强度量 $T$、$p$、$\mu_i$ 之间存在约束，只有 $k+1$ 个独立。

## 4.3 多元系的复相平衡条件

$\varphi$ 个相共存时：各相 $T$ 相等、$p$ 相等、每个组分在各相中的 $\mu_i$ 相等。

**吉布斯相律**：$f = k - \varphi + 2$

## 4.4 化学平衡条件

化学反应 $\sum_i \nu_i A_i = 0$ 的平衡条件：$\sum_i \nu_i \mu_i = 0$

## 4.5 混合理想气体的性质

道尔顿分压定律：$p = \sum_i p_i$，$p_i = \frac{n_i RT}{V}$

理想气体化学势：$\mu_i = RT[\phi_i(T) + \ln p_i]$

吉布斯佯谬：同种气体混合不产生熵增。

## 4.6 质量作用定律

对理想气体化学反应：$\prod_i p_i^{\nu_i} = K_p(T)$

范特霍夫方程：$\frac{\mathrm{d}\ln K_p}{\mathrm{d}T} = \frac{\Delta H_m}{RT^2}$

放热反应 $(\Delta H < 0)$ 的平衡常数随温度升高而减小。

## 4.7 理想溶液的化学势

拉乌尔定律（溶剂）：$p_A = p_A^* x_A$。亨利定律（溶质）：$p_B = k x_B$。

稀溶液溶剂化学势：$\mu_A = \mu_A^*(T,p) + RT\ln x_A$

稀溶液的依数性质：蒸气压降低、沸点升高、凝固点降低、渗透压。

# 第六章 近独立粒子的最概然分布

## 6.1 统计物理的基本概念

**微观状态**：由所有粒子的力学运动状态确定。**宏观状态**：由宏观参量描述，对应大量微观状态。

**等概率原理**：平衡态孤立系中，各微观状态出现的概率相等（统计物理基本假设）。

**热力学概率** $\Omega$：与宏观态对应的微观状态数。

**玻尔兹曼关系**：$\boxed{S = k\ln\Omega}$

## 6.2 粒子运动状态的经典描述

$\mu$ 空间：以粒子的 $2r$ 个坐标和动量为轴的 $2r$ 维空间，一个点对应一个粒子的运动状态。

相格大小：$\Delta q_1\cdots\Delta q_r\Delta p_1\cdots\Delta p_r \approx h^r$。

三维自由粒子态密度：

$$g(p)\mathrm{d}p = \frac{4\pi V}{h^3}p^2\mathrm{d}p$$

$$g(\varepsilon)\mathrm{d}\varepsilon = \frac{2\pi V}{h^3}(2m)^{3/2}\sqrt{\varepsilon}\,\mathrm{d}\varepsilon$$

## 6.3 粒子运动状态的量子描述

量子力学中粒子能量取分立值（能级）。**简并度** $\omega_l$：第 $l$ 个能级对应的不同量子状态数。

宏观态分布 $\{a_l\}$ 满足：$\sum_l a_l = N$，$\sum_l a_l\varepsilon_l = E$。

## 6.4 等概率原理

平衡态孤立系各微观状态等概率。分布 $\{a_l\}$ 出现的概率正比于其微观状态数 $\Omega\{a_l\}$。

## 6.5 玻尔兹曼分布

条件：近独立粒子，经典极限（$a_l \ll \omega_l$）。

微观状态数：$W = \frac{N!}{\prod_l a_l!}\prod_l \omega_l^{a_l}$

在 $\sum_l a_l = N$，$\sum_l a_l\varepsilon_l = E$ 约束下求最概然分布：

$$\boxed{a_l = \omega_l e^{-\alpha - \beta\varepsilon_l}}$$

其中 $\beta = 1/kT$，$e^{-\alpha} = N/Z_1$。单粒子配分函数：$Z_1 = \sum_l \omega_l e^{-\beta\varepsilon_l}$。

## 6.6 玻色分布和费米分布

**玻色子**（整数自旋，每个量子态可容纳任意多粒子）：

$$W_B = \prod_l \frac{(\omega_l + a_l - 1)!}{a_l!(\omega_l - 1)!},\quad a_l = \frac{\omega_l}{e^{\alpha + \beta\varepsilon_l} - 1}$$

**费米子**（半整数自旋，泡利原理，每量子态最多一个粒子）：

$$W_F = \prod_l \frac{\omega_l!}{a_l!(\omega_l - a_l)!},\quad a_l = \frac{\omega_l}{e^{\alpha + \beta\varepsilon_l} + 1}$$

当 $e^\alpha \gg 1$（经典极限）时，两种量子分布均过渡到玻尔兹曼分布。

## 6.7 三种分布的关系

| 分布 | 适用粒子 | 单态粒子数限制 |
|------|---------|---------------|
| 玻尔兹曼 | 可分辨/经典极限 | 无限制 |
| 玻色 | 玻色子（整数自旋） | 无限制 |
| 费米 | 费米子（半整数自旋） | $\leq 1$ |

# 第七章 玻尔兹曼统计

## 7.1 热力学量的统计表达式

单粒子配分函数 $Z_1 = \sum_l \omega_l e^{-\beta\varepsilon_l}$，$N = e^{-\alpha}Z_1$。

内能：$U = -N\frac{\partial\ln Z_1}{\partial\beta}$

广义力：$Y = -\frac{N}{\beta}\frac{\partial\ln Z_1}{\partial y}$，如 $p = \frac{N}{\beta}\frac{\partial\ln Z_1}{\partial V}$

熵：$S = Nk\left(\ln Z_1 - \beta\frac{\partial\ln Z_1}{\partial\beta}\right)$

自由能：$F = -NkT\ln\left(\frac{Z_1}{N}\right) - NkT$（利用斯特林公式）

## 7.2 理想气体的物态方程

单原子分子理想气体：$Z_1 = V\left(\frac{2\pi mkT}{h^2}\right)^{3/2}$

$$p = \frac{NkT}{V},\quad U = \frac{3}{2}NkT$$

**能量均分定理**：温度为 $T$ 的平衡态，每个平方项能量分量的统计平均值等于 $\frac{1}{2}kT$。

## 7.3 麦克斯韦速度分布律

$$f(\mathbf{v})\mathrm{d}^3v = \left(\frac{m}{2\pi kT}\right)^{3/2} e^{-\frac{mv^2}{2kT}}\mathrm{d}^3v$$

$$f(v)\mathrm{d}v = 4\pi\left(\frac{m}{2\pi kT}\right)^{3/2} v^2 e^{-\frac{mv^2}{2kT}}\mathrm{d}v$$

特征速率：$v_p = \sqrt{\frac{2kT}{m}} < \bar{v} = \sqrt{\frac{8kT}{\pi m}} < v_{rms} = \sqrt{\frac{3kT}{m}}$

## 7.4 能量均分定理及其应用

理想气体热容：
- 单原子：$C_V = \frac{3}{2}R$
- 双原子：$C_V = \frac{5}{2}R$（平动+转动）
- 双原子含振动：$C_V = \frac{7}{2}R$

经典能量均分定理无法解释低温下热容减小（需用爱因斯坦、德拜量子理论）。

## 7.5 固体的热容——爱因斯坦理论

$N$ 个独立谐振子（频率 $\omega$），谐振子能级 $\varepsilon_n = (n + \frac{1}{2})\hbar\omega$。

爱因斯坦热容公式：

$$C_V = 3Nk\left(\frac{\Theta_E}{T}\right)^2 \frac{e^{\Theta_E/T}}{(e^{\Theta_E/T} - 1)^2}$$

其中 $\Theta_E = \hbar\omega/k$。
- $T \gg \Theta_E$：$C_V \to 3R$（杜隆-珀蒂定律）
- $T \ll \Theta_E$：$C_V \propto e^{-\Theta_E/T}$（定性正确但低温定量偏低）

# 第八章 玻色统计和费米统计

## 8.1 热力学量的统计表达式

巨配分函数：$\Xi = \prod_l \Xi_l$

对玻色子 $\Xi_l = (1 - e^{-\alpha - \beta\varepsilon_l})^{-\omega_l}$，对费米子 $\Xi_l = (1 + e^{-\alpha - \beta\varepsilon_l})^{\omega_l}$

$$\ln\Xi = \mp \sum_l \omega_l\ln(1 \mp e^{-\alpha - \beta\varepsilon_l})\quad\text{（上玻色下费米）}$$

平均粒子数：$\bar{N} = \sum_l \frac{\omega_l}{e^{\alpha+\beta\varepsilon_l} \mp 1}$

内能：$U = -\frac{\partial\ln\Xi}{\partial\beta}$。熵：$S = k(\ln\Xi + \beta U + \alpha\bar{N})$

## 8.2 弱简并玻色气体和费米气体

$e^\alpha \gg 1$ 但不够大时，展开分布函数得一级修正：

$$pV = NkT\left[1 \pm \frac{1}{2^{5/2}}\frac{N}{V}\left(\frac{h^2}{2\pi mkT}\right)^{3/2} + \cdots\right]$$

$+$ 费米（等效排斥），$-$ 玻色（等效吸引）。

## 8.3 玻色-爱因斯坦凝聚

BEC 临界温度：$T_c = \frac{h^2}{2\pi mk}\left(\frac{N}{2.612V}\right)^{2/3}$

$T < T_c$ 时 $\mu = 0$，凝聚粒子数比例：$\frac{N_0}{N} = 1 - \left(\frac{T}{T_c}\right)^{3/2}$

BEC 是动量空间的凝聚，源于玻色子的量子统计性质，不依赖粒子间相互作用。

## 8.4 光子气体

光子 $\mu = 0$（光子数不守恒）。态密度：$\omega(\varepsilon)\mathrm{d}\varepsilon = \frac{V}{\pi^2\hbar^3 c^3}\varepsilon^2\mathrm{d}\varepsilon$

**普朗克公式**：

$$u(\nu,T)\mathrm{d}\nu = \frac{8\pi h\nu^3}{c^3}\frac{1}{e^{h\nu/kT} - 1}\mathrm{d}\nu$$

- $h\nu \ll kT$：$u_\nu \propto \nu^2 T$（瑞利-金斯）
- $h\nu \gg kT$：$u_\nu \propto \nu^3 e^{-h\nu/kT}$（维恩）

积分得 $U/V = aT^4$（斯特藩-玻尔兹曼定律）。

## 8.5 金属中的自由电子气体

费米-狄拉克分布：$f(\varepsilon) = \frac{1}{e^{\beta(\varepsilon - \mu)} + 1}$

$T = 0$ 时：$f(\varepsilon) = \begin{cases} 1, & \varepsilon < \varepsilon_F \\ 0, & \varepsilon > \varepsilon_F \end{cases}$

费米能级：$\varepsilon_F = \frac{\hbar^2}{2m}(3\pi^2 n)^{2/3}$。费米温度 $T_F = \varepsilon_F/k \sim 10^4\text{--}10^5\,\text{K}$。

$T=0$ 基态内能：$U_0 = \frac{3}{5}N\varepsilon_F$

低温电子热容：$C_V = \frac{\pi^2}{2}Nk\frac{T}{T_F} = \gamma T$

电子热容与 $T$ 成正比，因为只有费米面附近 $\sim kT$ 范围内的电子被热激发。

# 第九章 系综理论

## 9.1 相空间与刘维尔定理

$\Gamma$ 空间：由系统全部广义坐标和广义动量张成的 $2s$ 维空间。一个代表点对应系统的一个微观状态。

**系综**：大量结构相同、宏观条件相同的系统的集合。

**刘维尔定理**：$\frac{\mathrm{d}\rho}{\mathrm{d}t} = \frac{\partial\rho}{\partial t} + \{\rho, H\} = 0$。代表点在 $\Gamma$ 空间的运动不可压缩。平衡态 $\frac{\partial\rho}{\partial t} = 0$，$\rho$ 只是 $H$ 的函数。

## 9.2 微正则系综

孤立系（$N,E,V$ 确定，能量在 $E$ 到 $E+\Delta E$）。分布函数 $\rho = \text{const}$。

微观状态数：$\Omega(N,E,V) = \frac{1}{N!h^{Nr}}\int_{E\leq H\leq E+\Delta E}\mathrm{d}\Gamma$

$$S = k\ln\Omega$$

$$\frac{1}{T} = \left(\frac{\partial S}{\partial E}\right)_{N,V},\quad \frac{p}{T} = \left(\frac{\partial S}{\partial V}\right)_{N,E},\quad \frac{\mu}{T} = -\left(\frac{\partial S}{\partial N}\right)_{E,V}$$

## 9.3 正则系综

与大热源接触的闭系（$N,V,T$ 确定）。

正则分布：$\rho_s = \frac{1}{Z}e^{-\beta E_s}$

正则配分函数：$Z(N,V,T) = \sum_s e^{-\beta E_s} = \frac{1}{N!h^{Nr}}\int e^{-\beta H}\mathrm{d}\Gamma$

$$\boxed{F = -kT\ln Z}$$

内能：$U = -\frac{\partial\ln Z}{\partial\beta}$，压强：$p = \frac{1}{\beta}\frac{\partial\ln Z}{\partial V}$，熵：$S = k(\ln Z + \beta U)$

能量涨落：$\overline{(E - \bar{E})^2} = kT^2 C_V$，相对涨落 $\sim 1/\sqrt{N}$。

## 9.4 巨正则系综

与大热源兼粒子源接触的开系（$\mu,V,T$ 确定）。

巨正则分布：$\rho_{N,s} = \frac{1}{\Xi}e^{-\alpha N - \beta E_s(N)}$，$\alpha = -\mu/kT$

巨配分函数：$\Xi(\mu,V,T) = \sum_{N=0}^{\infty} e^{\beta\mu N}Z(N,V,T)$

$$\boxed{J = -pV = -kT\ln\Xi}$$

平均粒子数：$\bar{N} = -\frac{\partial\ln\Xi}{\partial\alpha}$。粒子数涨落：$\overline{(N-\bar{N})^2} = kT\left(\frac{\partial\bar{N}}{\partial\mu}\right)_{T,V}$

## 9.5 三种系综的等价性

在热力学极限（$N\to\infty, V\to\infty, N/V$ 有限）下，三种系综等价。

| 系综 | 固定量 | 特征函数 | 分布 |
|------|--------|---------|------|
| 微正则 | $N,E,V$ | $S = k\ln\Omega$ | $\rho = \text{const}$ |
| 正则 | $N,V,T$ | $F = -kT\ln Z$ | $\rho \propto e^{-\beta E}$ |
| 巨正则 | $\mu,V,T$ | $J = -kT\ln\Xi$ | $\rho \propto e^{-\alpha N - \beta E}$ |

实际计算优先级：巨正则 > 正则 > 微正则。

# 第十章 涨落理论（基础概念）

## 10.1 涨落的基本概念

**涨落**：宏观物理量围绕其平均值的随机偏离。

爱因斯坦涨落公式（孤立系）：$W \propto e^{\Delta S/k}$。对闭系（等温）：$W \propto e^{-W_{\min}/kT}$，$W_{\min}$ 为涨落所需最小功。

## 10.2 布朗运动

悬浮微粒因分子不平衡碰撞而做无规则运动。

爱因斯坦关系（一维）：$\overline{x^2} = 2Dt$

斯托克斯-爱因斯坦关系：$D = \frac{kT}{6\pi\eta a}$

布朗运动是涨落的直观体现，分子热运动的直接证据。
